在第十三章中，我们终于有机会将快速职业手册课程中学到的所有知识整合起来。众所周知，成为一名成功的扑克玩家最困难的部分之一就是弥合理解与应用之间的差距。我对这节课最大的希望是让你全面思考扑克的运作方式，以及如何从头到尾拼凑出一手牌。我真诚地相信，如果你能养成逻辑思考牌局的习惯，那么无论情况如何，你最终都能轻松地分析任何牌局或局面。

读牌和了解对手的能力是最难教授的技能之一，部分原因是学习它的唯一方法是投入大量时间和积累经验。但说实话，如果我必须在成为世界级的 PLO 理论专家和世界级的读牌高手之间做出选择，我随时都愿意选择读牌技能。

### 概述

在深入探讨一些更高级的概念之前，让我们先快速概述一下河牌圈的玩法，并进行一些概括。首先，或许也是最重要的一点，请记住**你在河牌圈的所有决策主要都基于数学**。举个简单的例子，假设你在河牌圈持有抓诈唬牌，而对手决定向你下注一个底池大小的注。无论你如何拼凑他的范围，一个简单的事实是，由于你得到的赔率是 2:1，你必须有超过 33% 的胜率才能在跟注时获利。

此外，几乎所有河牌圈决策都基于数学的主要原因是，牌面价值是完全静态的，这意味着每个玩家的范围要么是坚果牌，要么是空气牌。毕竟，既然没有剩余的牌，我们就不再需要担心保护自己的牌、对抗范围的权益，或者进行多街牌规划之类的事情了。简而言之，你要么拿着最好的牌，要么没有。我们稍后会看到，这比我描述的要复杂一些，但目前这是一个很好的起点。

如果你还记得之前的课程，我们说过，在玩家范围两极分化的静态公共牌面上，正确的决策主要基于频率和对手的倾向，所以玩家倾向和数学是决定你能否成功渡过河牌圈的两个决定性因素，这并不奇怪。

我发现一个可靠的经验法则是，级别越低，玩家在河牌圈通常越直接。这通常也适用于所有街道，但在河牌圈尤其如此。随着牌局的推进，玩家的范围会缩小，大多数低级别玩家无法很好地阅读牌局，无法发挥创造力，比如在河牌圈过牌 - 加注以获得价值，在河牌圈过牌 - 加注诈唬，或者把超对或其他弱成手牌变成诈唬。坦白说，大多数时候他们的牌看起来和实际情况一模一样。正如我们将在本章中探讨的那样，这让我们能够运用适当的下注尺度来剥削那些不愿适应我们策略的玩家。

### 河牌圈下注尺寸

既然我们显然不能在河牌圈半诈唬，那么价值下注和诈唬就能让我们赚得盆满钵满。那么我们应该选择什么样的下注尺寸呢？需要考虑的三个主要因素是牌面结构、行动和对手的倾向。为了让你大致了解每种下注尺寸在河牌圈下注范围中的位置，我认为最好快速总结一下每种下注尺寸最常用的情况以及对手是如何解读的。

简单来说，河牌圈下注尺寸可以分为三类：非常小、中等或非常大。非常小的下注（10-30% 底池）通常用于引诱弱牌跟注，与有交牌历史对手的阻挡下注进行平衡，或者引诱激进对手进行轻的加注。

刚才我们提到，玩家在 PLO 的河牌圈通常比在其他游戏中玩得更直接。这种策略最能发挥你优势的场合之一就是河牌圈，当你的对手在河牌圈行动时，很明显你的对手只有一手失败的听牌或一手很弱的成手牌。下注过大会导致他们更快地弃牌，但小额下注通常能让你榨取一些额外的利润，而不是过牌后一无所获。此外，尤其是在低级别，玩家们非常讨厌面对河牌圈的小额下注下弃牌，而且由于他们玩法直接，很少看到小额下注诱使对手轻的过牌 - 加注。大多数时候，他们甚至不会考虑过牌 - 加注，他们只会做出跟注还是弃牌的决定。

我观察到，苦苦挣扎的低级别或中级别玩家与成功的中级别或高级别玩家之间的另一个区别在于，他们做出漂亮弃牌的能力。大多数人认为成功的扑克玩家都是激进的、几乎无所不能的强者，他们的大部分资金都来自诈唬，但这根本不是事实。如果你有机会观察高额玩家的玩法，你会发现，无论他们拿到多好的河牌跟注赔率，如果他们知道自己输了，就会毫不犹豫地弃牌。幸运的是，我们大多数时候都不会和高额职业玩家对战，所以在考虑小额价值下注时，一定要记住这一点。

从长远来看，在河牌圈小额下注并偶尔被反拿价值，比因为害怕被更好的牌跟注而过牌要赚得更多。简而言之，**在低级别价值下注的场合，宁可为了薄价值下注，也不要过牌。** 此外，一旦你在几次场合被反拿价值，你就能通过反复试验，很快学会哪些场合应该过牌而不是下注。

中尺寸下注指的是 40-70% 底池。这是你默认的河牌圈下注尺寸，大多数情况下，这种下注尺寸用于在听牌全部错过的情况下，主动进行持续下注。中尺寸下注是合理的，因为它足以在听牌失败时提供良好的弃牌权益平衡，或者在你拿到最佳牌时提供可靠的价值。

最后，大尺寸下注指的是 75-100% 底池。在河牌圈全下的最常见场合之一是你需要最大化弃牌权益。对抗某些对手时，你下注的金额不会改变他们的跟注频率，要么是因为你牌面形象很差，他们愿意用任何牌跟注，要么是因为他们只是喜欢烧钱。对抗其他对手时，你选择的下注尺寸会显著影响他们的跟注频率。

例如，当你知道对手拿着弱的抓诈唬牌（比如在四连接顺子牌面拿到弱两对）时，你不会想用空气牌下注 25% 底池，因为你给他的弃牌赔率太好了。相反，下注大，迫使他需要更高的判断正确率，因此当他判断错误时，损失的钱就越多。

当然，只有在试图迫使对手弃牌时才下大注并非明智之举。那么，我们应该在哪些情况下进行价值下注呢？**最佳的价值下注时机是当你确信自己的牌力最佳，并且对手的跟注范围缺乏弹性时。** 尤其当你的牌面形象不佳，或者坦白地说，你任何时候认为对手拿着抓诈唬牌时，考虑下更大的价值下注都是一个好主意。记住，让你的权益翻倍的最快方法之一，就是加倍你的价值下注。

### 价值下注

承接上一节的讨论，我必须提醒各位：大部分盈利并非来自河牌圈设陷阱。正如我们已论证的，玩家在河牌圈的打法往往过于直白，这意味着我们必须主动承担起往底池下注资金的责任，防止对手用那些愿意跟注但牌力更差的牌选择随后过牌。

没有什么比让对手用他们本来会跟注的牌随后过牌更糟糕的了，在价值下注的场景中，这无疑是我们最想避免的情况。我不会花太多时间讨论如何在你拿着超强牌的时候进行价值下注，因为我认为这对你来说很容易自学，所以让我们专注于在进行薄价值下注时需要考虑的事项。

对手摊牌的频率与你进行薄价值下注的力度直接相关。对抗更紧的玩家，目标是两极化你的范围。为了解释为什么这是一种有效的策略，让我们快速回顾一下我们在第九章中学到的几个概念。回想一下我们是如何确定你的翻牌前牌决定了你有多少翻牌后权益的。简而言之，更紧的玩家拥有更多的翻牌后权益，因为他们比更松的玩家玩更多的更大、同花性、连接性更强的牌。

这个概念也贯穿到了河牌圈。当更紧的玩家摊牌时，你可以指望他们比更松的玩家更频繁地拿到坚果牌或接近坚果牌的牌。虽然他们的摊牌率（WTSD）低于平均水平，但他们的摊牌胜率（W$SD）通常高于平均水平。这是因为更紧的玩家比更松的玩家拥有更多 A 同花、坚果顺子听牌和更好的两对组合。

这就是为什么我亲身经历以及我学生身上最常见的被反拿价值的情况，都是在对紧弱玩家进行过薄的价值下注时发生的。简而言之，如果你对紧弱玩家下注或加注，要意识到你要么 [A] 把一手牌变成诈唬，[B] 什么牌都没有，要么 [C] 手里有超强牌，宝贝！

另一方面，我们真正希望进行薄价值下注的玩家是那些摊牌率百分比高或摊牌胜率低的较松玩家。面对较弱的玩家（或任何玩得太松的玩家），我们的目标是惩罚他们玩太多手牌。就像我们想通过更多地诈唬来惩罚紧手玩家一样，我们也想通过薄价值下注来惩罚松手玩家跟注过多。

这些概念的目的是让你将它们用作指导方针，而不是法律。关注人们摊牌的频率将使你更清楚地了解他们的手牌强度，但牢记牌面结构和与对手的动态也很重要。

### 诈唬

让我们来看看价值下注的异装兄弟：诈唬。需要弥补的两种主要诈唬类型是：我们应该把成牌变成诈唬的时机，以及纯粹用空气牌诈唬 - 加注。

把成牌变成诈唬是我在扑克中最喜欢的事情之一，因为如果操作正确，它可以很好地证明你的牌被正确解读了。把一手牌变成诈唬意味着你的牌包含一些绝对价值（比如在顺子牌面上拿到暗三条，或者在对子牌面上拿到弱同花），但相对于对手的范围没有任何相对价值。

有经验的玩家和不输不赢或输钱的玩家之间一个显著的区别是，后者通常会用他们应该弃牌或变成诈唬的牌跟注。简而言之，由于他们的读牌能力不够敏锐，他们无法识别自己的牌无法战胜任何对手的情况。一旦你玩了一定量的手牌并掌握了技巧，将成手牌变成诈唬牌就变得显而易见且容易得多。就像任何其他诈唬牌一样，在扣动扳机之前需要考虑的主要因素是，你的牌的线路是否与你代表的牌相符，以及你是否拥有足够的弃牌权益来获利。

对于诈唬 - 加注来说，最佳的河牌圈攻击方式是听牌完成的河牌圈。在 PLO 游戏中，玩家比在其他游戏中更担心碰到坚果牌，因此更容易迫使对手弃牌。最常见的利用听牌完成的河牌圈诈唬 - 加注获利的情况与翻牌后战争中概述的许多概念相关。

例如，在动态牌面中，一个主要策略是在翻牌圈用一个听牌在有利位置跟注，并在转牌圈代表另一个完成的听牌，从而迫使对手放弃他的牌。这个策略也适用于河牌圈。例如，一种常见的情况是，对手在翻牌圈和转牌圈下注底池代表顺子，然后河牌圈完成了同花或形成了对子。这种情况下，你很容易诈唬并赢得底池。

之前，我们明确了河牌圈的范围是静态的，因此是两极化的。另外，回想一下，在牌面学校的课程中，我们讨论过，当范围两极化时，没有必要为了诈唬 - 加注下注满池。这意味着，调整你在河牌圈的下注大小可以显著提高你诈唬的盈利能力。

最后，当你看到对手在河牌圈抛出一个弱注或阻挡型下注时，你应该强烈考虑对他们进行诈唬 - 加注。从读牌的角度来看，大多数对手都不够老练，无法用小额下注来引诱你诈唬 - 加注。例如，如果一名弱牌手在河牌圈领先下注 1/3 底池，并且你知道他们没有能力进行薄价值下注或小额下注来诱使加注，那么你可以轻松地将他们的范围缩小到大部分是低强度到中等强度的牌，或者完全是空气牌。

### 过牌 - 加注

既然你已经在快速职业手册中学到了这，想必知道我非常热衷于尽可能多地过牌 - 加注。过牌 - 加注河牌圈有很多好处，而且和我们之前讲过的其他所有牌局一样，过牌 - 加注通过限制对手的可选应对方案，让你更难对付。

价值下注就是一个最好的例子。如果对手知道你有能力在河牌圈轻的过牌 - 加注，他就无法像他希望的那样对你进行薄价值下注。这对我们来说是一个很大的优势，因为最终这意味着我们可以用弱牌或中等牌力的牌更便宜地摊牌。**在决定是否进行过牌 - 加注，无论是为了价值还是为了诈唬时，了解对手能接受多弱的价值下注尺寸或许是最重要的考量因素。** 此外，根据 SPR 的不同，在河牌圈过牌 - 加注会让你有机会进行最后的下注。

在河牌圈过牌 - 加注时，需要遵循一些关键点，它们将帮助你选择合适的时机进行诈唬，或最大化你手中牌的价值。**就过牌 - 加注的价值而言，最佳时机是对手会用他跟注过牌 - 加注的牌进行价值下注的时候。** 同时，如果你持有价值牌，你最应该避免的时机是过牌 - 加注过薄，因为只有比你更好的牌会跟注你。稍后，我们将进一步讨论一个有利可图的过牌 - 加注场景，即翻牌后的行动使你的最强范围大于他的最强范围。

### 三次开枪

在上一章中，我们详细讨论了有利可图的两次开枪时机所需的要素。三次开枪和两次开枪有一些相似的特征，但它们之间的差异足以让我们有必要分别进行探讨。

现在，在考虑三次开枪时，主要需要考虑三个方面。首先，也是最重要的一点，是对手的倾向。通常情况下，在单调牌面面对不愿放弃两对的跟注站时，全下是个坏主意，但这不应该让你感到惊讶，因为我们在整本书中都详细讨论了这一点。

接下来，我们必须分析对手的范围与目前的行动如何匹配。问自己一些问题，例如 “他不可能有什么牌？” ，“他总是这样玩什么牌？” 你甚至可以更具体地问 “在这个牌面他会如何玩暗三条？” 或者 “什么样的牌会在这里跟注看牌？” 此外，只需想想有多少牌型组合能打败你，就能看出用对手抓诈唬的盈利能力有多强。

关于三次开枪，第三个值得考虑的方面是对手的范围如何与牌面结构互动。不仅要注意河牌圈的坚果牌，牌面结构在牌局中的发展也对我们利用对手的能力有很大影响。

最后，或许也是最重要的一点，在向底池连续三次开枪时需要考虑的是，**如果你从未被抓到诈唬，那么你可能诈唬得不够**。我有几个学生在进行第三次开枪时遇到了很多麻烦，这彻底毁掉了他们的非摊牌赢利，也降低了他们的胜率。就像我之前说过的，积累大底池然后又放弃，会摧毁你的胜率，也会摧毁你的信心。没有什么比在三次开枪时胆怯地放弃，结果却看到对手亮出一手弱的一对，即使你只是念念有词 “下注”，这手牌对手也会毫无疑问的弃牌。

在上一节中，我解释过大多数玩家在河牌圈诈唬的频率远远低于他们应该的频率。如果你问我，我认为他们犹豫是否进行第三次开枪并非因为害怕或其他原因，而更多的是因为他们没有意识到他们的下注需要多频繁地奏效才能盈利。这就是为什么我决定提供表格 13.1，它显示了我们的对手需要多频繁地弃牌才能让我们的最后一次下注盈利。

表格 13.1 需要的弃牌频率。

| 下注大小（底池百分比） | 需要的弃牌频率 |
| --- | --- |
| 50% | 33% |
| 60% | 38% |
| 70% | 41% |
| 80% | 44% |
| 90% | 47% |
| 100% | 50% |

这实际上只是 PLO 快速权益统计倒数第二页的表格。在最左侧一列，你会看到我们的权益被恰当地标示为百分比，但主要需要考虑的变量是下注额（以底池的百分比表示）以及为了形成有利可图的诈唬所需的弃牌频率。例如，如果我们在河牌圈诈唬一个满池的下注，对手需要弃牌一半的次数才能盈利。同样，如果我们下注 50% 的底池，那么只需要 1/3 的次数有效就能盈利。

记住这张表格，然后在决定再次下注之前，问问自己，你是否认为你的下注能够经常有效并带来盈利。例如，“如果我在这里下注满池，他会有一半的次数弃牌吗？” 如果不是，那么你或许应该放弃。要回答这个问题，你需要知道我们有多少弃牌权益，所以分析河牌如何击中对手以及你被感知的范围是关键。最后，考虑到你的牌型和历史，考虑一下对手会如何应对你的下注大小。

### 读牌

在本章前面，我提到读牌是迄今为止任何扑克玩家都能拥有的最宝贵的技能。坦白说，你可能花费数年时间观看所有扑克视频，研读所有可用的扑克书籍，却仍然在街上破产。归根结底，除非你知道如何将理论应用于牌桌，否则了解所有理论毫无意义。我指的是能够解读牌局，了解对手的能力，并有勇气坚持你的解读。

每位玩家在分析任何一手牌时都会经历自己独特的内在过程，所以关于读牌，我认为你能为自己做的最好的事情就是专注于继续使用最适合自己的方法。当你与其他玩家接触时，试着学习他们分析牌局的方法，看看你是否能吸收他们擅长的一些技巧，让你也能更轻松地读牌。

过去，我发现几乎每位正在努力提升自己牌技的玩家至少在一件事上做得非常出色。这就是为什么我一直把学习他们比我强的地方作为我的目标，然后将其融入到我的游戏中。这样，你就可以从遇到的每位玩家身上汲取所有优秀特质。最终你会发现，坚持这样做一段时间后，你最终会拥有一场扎实、全面的游戏。虽然我建议你通过大量投入来尝试，但我还是想提一下过去几年里帮助我提升读牌能力的几点建议。

第一步是反向玩牌。这本质上意味着从头到尾拼凑对手的范围，看看他的故事是否合理。反向玩牌就像扮演侦探的角色，目标是尽可能缩小他们的范围。记住，玩家在河牌圈的牌实际上只能是价值下注、抓诈唬牌和空气牌。如果你能缩小这个范围，你就能获得更准确的信息来做决定。这与我们关于三次开枪的讨论息息相关，我们的目标是养成经常问自己诸如 “他总是这样玩什么牌？他永远不可能有什么牌？” 之类的问题的习惯。

其次，或许也是最重要的一点，了解对手的能力。这几乎决定了你在河牌圈所做的一切，从你能用多薄的价值下注或诈唬，到你随后过牌并试图摊牌的牌。

最后，从对手在每个级别下注时掌握的下注大小和时机信息中，你可以获得大量信息。级别越低，下注大小信息往往越明显。我发现最可靠的时机信息是快速行动。**河牌圈的快速行动几乎总是倾向于要么下坚果牌，要么什么都没有。** 想想看：在考虑薄价值下注时，需要花时间回顾牌局，考虑选择哪个下注金额，然后最终在屏幕上点击下注。

了解河牌圈的快速行动是两极化的，可以让读牌变得容易得多，因为这会把中等牌力的牌从他们的范围中剔除。正如我们之前多次讨论过的，玩家范围两极化的场合取决于频率和对手的倾向。当玩家在河牌圈快速下注时，你的任务就是根据之前的行动和牌面结构，判断他们是否有能力在特定场合诈唬。

### 第十三章 概念小结

1. 记住，让胜率翻倍的最快方法之一是加倍你的价值下注尺寸。最佳的重注价值时机出现在你面对一个跟注范围受限的对手，而你又自信自己的牌力最佳时。
2. 大多数河牌圈决策都基于数学。这是因为手牌价值是静态的，因为没有更多街牌可玩，所以只剩下价值下注和诈唬。我们可以得出的另一个结论是，大多数玩家在没有坚果牌时倾向于河牌圈玩的直接。这可以让你利用下注尺寸来更好地剥削对手。
3. 我们可以将下注尺寸分为三类：小额（10-30% 底池）、中额（40-70% 底池）和大额（75% 底池以上）。小额下注通常用于引诱对手跟注、平衡阻挡下注以及引诱激进对手的诈唬加注。当许多听牌都失败时，我们默认的下注额是价值下注或诈唬。面对受限跟注范围时，大额下注可以用来最大化弃牌权益。或者，当你确信自己拿着最好的牌，但面对弹性跟注范围时，大额下注可以用来最大化价值下注。
4. 对手摊牌的频率与你进行薄价值下注获利的程度直接相关。面对低摊牌率（WTSD）的紧手玩家，应该使用两极化范围下注。面对高摊牌率（WTSD）的松手玩家，应该使用更宽或更薄的价值下注。
5. 高水平的玩家擅长的一件事就是将他们的牌变成诈唬。当你的牌具有绝对价值，但相对于对手的范围没有相对价值，并且你的下注路线与你想要表现的一致时，你应该考虑这一点。
6. 河牌圈过牌 - 加注会让你更难对付，因为对手无法对你进行薄价值下注。此外，它让你进行最后下注，从而获得一些弃牌权益。每当你的对手会用他愿意跟注过牌 - 加注的牌进行价值下注时，就过牌 - 加注。如果是为了价值下注，如果更差的牌不会跟注你，就不要过牌 - 加注。
7. 三次开枪时要考虑的三个主要因素是对手的倾向、对手的范围以及该范围与牌面结构的相互作用。避免对跟注站进行三次开枪，或者当击败你的牌组合太多时，尤其是在牌面结构发展对对手范围有利的情况下。
8. 读牌时，反向玩牌以缩小他的范围。然后问问自己，他用范围中的每个部分能做什么，他有多大可能将一手牌变成诈唬牌，或者进行薄价值下注？最后，考虑下注大小和时机的暗示，比如快速行动往往代表着一个两极化的范围。
