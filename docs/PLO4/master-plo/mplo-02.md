# 第二章 整体游戏计划

在前几章中，您了解了博弈论、游戏框架、游戏选择和牌桌选择的重要战略含义，以及两种帮助您制定制胜策略和游戏计划的博弈论方法。在对抗较弱的对手时，使用剥削性策略来最大化您对对手漏洞的预期。平衡的、博弈论最优的方法在对抗强势玩家时效果更佳，因为它不会留下任何弱点和漏洞可供利用。

现在是时候更具体地考虑您的整体游戏计划了。如上所述，理论上的整体游戏计划会为牌桌上所有可能的情况提供明确的战略建议，包括范围考虑、起手牌选择、牌面结构、下注频率等等。接下来的章节将深入探讨这些主题。

人类的学习方式是首先理解原理，然后通过评估各种场景 (不像计算机那样计算所有可能的结果) 以某种直觉的方式应用它们。因此，我们将首先从理论的角度来探讨策略。然后，我们将讨论一些实际问题，例如游戏类型、对手、牌桌动态和下注结构，然后再讨论最重要的话题——如何玩这个游戏。在此之前，我们将讨论翻牌前理论和起手牌选择，然后讨论不同类型的公共牌，以及一些对任何 PLO 玩家都有用的原则和概念。

理论框架对于制定合理的游戏计划至关重要，因为它定义了游戏的结构，无论你的牌是什么。你希望在不同的公共牌结构和不同情况下，用你的全部范围形成一套模式和标准玩法。

游戏计划的第一步是创建完整的范围，尤其是你对抗与你级别相同级别典型对手的范围。游戏计划首先是一种理论方法，因此你必须确定一个适合你级别的整体策略。当然，并非所有玩家的风格都相同，但总的来说，你必须根据你级别标准玩法来规划你的牌局。你的整体策略应该能够对抗各种类型的对手。之后，你可以调整你的标准玩法，以应对不同的对手以及他们的特殊玩法和漏洞。

显然，在小额 PLO 游戏中，一个稳健的游戏计划在高额游戏中可能效果不佳，因为像激进、连续下注和诈唬频率这样的参数根本就来自不同的领域。高额玩家在低额游戏中也会遇到类似的困难。你需要调整你的策略以适应你正在玩的级别。

拥有针对典型情况的标准模式将对你的游戏大有裨益，因为你可以将注意力集中在更复杂的场合。游戏计划将帮助你更轻松、更有利可图地做出决策。游戏计划应该适用于你的整个范围，而不仅仅是一手牌。第一步是确定你自己的范围和基本游戏计划，即如何在某些标准情况下进行游戏。然后，你可以根据需要调整你的游戏计划以应对不同类型的对手。

你必须决定在每个位置的起手牌，以及在不同情况下如何继续游戏。这正是你的游戏计划的目的：以 GTO 的方式为标准位置设定标准范围。要定义精确的范围，你必须从总体上思考起手牌。什么样的牌才算好玩？在下一章关于翻牌前理论的章节中，你将学习一些原则，帮助你评估起手牌的质量以及它们在不同情况下的表现。由此，你可以制定出一些有用的标准策略，尤其是在多桌游戏中。
