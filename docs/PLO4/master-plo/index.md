# 精通 PLO：现代激进策略

## 引言

本书旨在阐释当今 PLO 游戏的扎实激进策略，提供每位玩家所需的基本理论知识。

本书分为理论和实践两部分。由于 PLO 游戏极其复杂，仅凭一些通用原则和博弈论基础知识是无法真正理解它的。理解理论基础固然重要，但你必须在时间压力下将知识运用到具体情境中，而这正是大多数玩家面临的挑战。

本书第一部分包含理论知识，帮助你从原理上理解游戏，并制定合理的游戏计划。我们将首先简要介绍博弈论及其对当今游戏的重要性。除了剥削性策略之外，我们还将介绍和解释博弈论最优 (GTO) 策略的概念，并概述两种基本的博弈论战略方法。这两种策略都基于良好的决策，但目标各不相同。第一种方法旨在利用对手的漏洞，并在特定场合最大化价值，而 GTO 方法则旨在构建均衡的范围和难以被利用的下注频率，防止对手利用你的策略获得预期价值 (EV)。在讨论了几个博弈论原则，例如位置意识、实现权益和计算最佳诈唬频率之后，我们将进入理论章节的核心：游戏计划。

我们将介绍并讨论整体游戏计划的概念，作为针对所有可能情况的明确战略建议，包括牌型选择原则 (位置、权益、可玩性和权益分布) 以及对即将进入的翻牌后局面的预测 (对手数量、筹码底池比、主动权、对手倾向) 。这些概念对于你的起手牌选择至关重要，并能帮助你构建合理且有利可图的范围。我们将详细讨论这些范围，从标准的开池和跟注范围开始，然后是高级激进的 3-bet 和 3-bet - 跟注范围，最后以合理的 4-bet 范围结束。各类起手牌 (大口袋对子、连牌、两对) 对抗不同范围的权益和权益分布表，将为您提供制定平衡翻牌前策略所需的数学基础。

除了了解翻牌前打法和起手牌选择的原则外，了解 PLO 翻牌后打法的关键方面也至关重要：牌面结构。要制定可靠的游戏计划，您必须评估不同类型的牌面，包括同花牌、连接性和手牌等级，以及它们相应的动态。您必须采用不同于干燥或静态牌面的下注策略来应对湿润、听牌较多的牌面。理解为什么应该在听牌较多的牌面下更大的注，在静态牌面以更高的频率持续下注，或者在面对被动或激进的对手时，在有利位置或不利位置使用不同的策略，将显著提高您的游戏水平。解读不同类型牌面的另一个关键点在于它们与各种翻牌前范围的互动。当你的范围在特定翻牌面无法代表很多强牌时 (例如，你的 3-bet 范围很紧，在 8-7-6 牌面时)，你必须调整你的下注和过牌范围来对抗更优秀的玩家，以避免被剥削。完美平衡的翻牌前范围会考虑到这一点，提供完整的翻牌面覆盖，以避免在特定翻牌面被剥削。

第一章末尾的心态部分应该提醒每位玩家，除了技术和策略性的游戏技巧外，控制情绪和失控，以及职业和资金管理，对于任何想要精通扑克游戏的玩家来说都是必不可少的。

第二部分将介绍实用方法，并结合牌例深入探讨典型情况，提供具体的建议，教你如何解读牌面结构、对手的范围，以及如何以平衡且结构化的方式发展自己的下注、加注和弃牌范围。不仅要了解和理解游戏的理论框架，还要将其应用于牌桌上，这一点至关重要。除了深入分析牌局范围和平衡的下注线路外，针对溜入、单次加注、3-bet 和 4-bet 底池等典型情况的牌例还将向每位雄心勃勃的玩家展示如何提升自己的牌局分析能力。我们将强调位置意识以及博弈论原理，以定义正确的翻牌后下注线路。

在最后两章 “如何适应封顶 (CAP) 游戏” 和 “深筹码打法” 中，我们强调了不同筹码量下的游戏。理解短筹码或深筹码如何影响你的战略方法以及诸如权益等概念的应用至关重要。虽然在 CAP 游戏中，你应该更多地基于权益优势而非翻牌后的可玩性来选择牌型 (这显而易见，因为翻牌后可玩性很小)，但在深筹码玩法中，位置、可玩性和欺骗性等概念更为重要。

将本书分为两部分是一个艰难的决定。我们这样做是因为我们坚信，将理论和实践分开对待对读者更有益。对所有可能的情况进行冗长而抽象的理论讨论会难以阅读，并且会淡化每个玩家都应该了解的那些扎实的原则和理论。另一方面，分析具体情况不仅能帮助读者理解如何以整体的策略来打实际牌局，还能展示如何应用战略和博弈论原则。
