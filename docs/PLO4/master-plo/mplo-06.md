# 第六章 心态

扑克心理学并非本书的重点，因此本章略显简短。但这并不意味着心理学在扑克游戏中只是边缘部分。心态对你的胜率至关重要，尤其是在当今的游戏中。近年来，PLO 玩家的技术水平有了很大的提升。与常客玩家的对局往往取决于心理稳定性和心态，而非技术水平。谁能更长时间地保持 A 状态，并且不陷入 B 或 C 状态，谁就能获得更多利润。

在接下来的章节中，我们将讨论情绪、牌局的管理、职业生涯管理以及至关重要的资金管理。每位玩家都应该深入研究这些主题。其他网站上也有一些优秀的资料，我们鼓励你阅读并了解更多。

### 情绪

扑克玩家花费大量时间研究策略以提高他们的成绩。当我们考虑这些非常数学化和技术性的问题时，我们常常会忘记游戏中最重要的一个方面。扑克是人玩的，其中涉及很多情绪。赢得巨额底池你会感到兴奋，爆冷反超你会感到高兴，输给小鱼你会生气，没中翻牌或总是遇到坚果牌你会失控。

情绪不仅仅是正确玩扑克的附带副作用。我们不仅仅是机器。我们必须接受，扑克和情绪是形影不离的。当我们顺风顺水时，我们会忽略这个事实，但当你处于下风期时，情绪的影响就会变得非常明显。你还记得上次因为害怕四条而没有加注坚果葫芦是什么时候吗？你上次因为认为对方不可能一直有牌而跟注，尽管你知道他很少诈唬，是什么时候？

大脑倾向于记住坏牌和负面情绪，而忘记你赢得抛硬币或免费听牌的美好时光。当你用被压制的暗三条全押输掉时，你会认为这是运气不好。同样的情况也发生在你用 60% 的权益下注却输掉的时候。你是领先热门，但你仍然有 40% 的输牌概率，而且当你输掉这些牌时，你会觉得对手把你彻底打败了。这些负面情绪会持续威胁你的 A 状态。

了解这种机制的运作方式至关重要。情绪的力量如此强大，它们可以迅速压倒你的思考能力。所有坏牌带来的情绪会在你的记忆中积累，并开始掌控你的决策。尤其是在长期的下风期，你可能会陷入一种自动驾驶模式。你的负面情绪占据上风，让你忘记了 A 状态，只是漫无目的地乱打。你失去了控制，不再理性思考。换句话说，你情绪失控了。控制和理解是掌控扑克游戏中情绪问题的关键。

为什么我总是发生这种情况？这不可能再发生了！我真是太倒霉了！如果你陷入这样的想法，你就处于失控状态，最终你会玩得一塌糊涂，甚至更糟。失控的形式多种多样，有些相当微妙。有一种 “坏运气失控”，是由于在太短的时间内连续爆冷输掉而导致的。还有一种 “不公正失控”：你总是拿着第二把坚果牌输，这 “太不公平了”。你会因为自己犯了一个愚蠢的错误而生气，导致 “错误失控”，进而犯下更多错误。你会陷入 “权利失控”，因为你认为自己是最棒的玩家，不应该输掉任何一手牌。

摆脱情绪失控的关键并非只是忽略情绪，并指望问题会在下一局牌局中得到解决。你必须了解负面情绪如何影响你的游戏，以及如何解决这些问题。第一步是时刻保持警觉，确保自己不会失控。一个好方法是分析导致你失控的典型情况，并在这种情况再次发生时自我监控。控制是解决情绪问题的关键。

在玩扑克时，累积的情绪会带来真正的风险，因为它们会严重影响你的决策质量。不要忽视它们，也不要试图立即解决问题，因为你无法做到。你必须逐步降低情绪的影响和强度，同时努力提高掌控局面的能力。最重要的是，要意识到自己何时会进入一种严重影响游戏质量的心态。那么唯一的解决办法就是停止游戏，重新集中注意力，为下一场牌局做好准备。这样做虽然可以保护你的资金，但并没有彻底解决问题。

这就是为什么你必须明白，负面情绪和失控并非源于爆冷输掉。当玩家认为自己遭受了不公平的对待时，他们就会失控。有些人认为输给弱手是不公平的。另一个例子是，在 95% 权益的情况下输掉牌局的痛苦经历。这看起来可能非常不幸和不公平，但最终，这是一个统计结果，每 100 手牌中就会出现 5 次。

为了更好地管理情绪——尤其是失控——你必须全面地看待一局牌中发生的所有事情。这可以帮助你接受爆冷出局和逆袭反杀是游戏的正常现象。嘿，有时阳光明媚，有时下雨。这是很自然的。在牌桌上管理情绪的关键在于调整和纠正你对一局牌中发生的事情的处理方式。接受负面结果是游戏的正常组成部分，将有助于你保持专注和冷静，因为你已经消除了导致失控的不切实际的期望。

围绕你的扑克游戏建立一个稳定的框架来平衡你在某些场次牌局中可能经历的所有沉重情绪是非常有益的。这引出了牌局管理的根本重要性。接下来我们将讨论如何利用牌局管理和时间管理来更好地管理你的情绪。记住，几乎不可能将情绪降到零，因为我们是人，而人是有情感的。你的目标是接受并控制你的情绪，这样它们就不会影响你发挥最佳水平的能力。

### 牌局管理

牌局 (Session) 管理至关重要。大多数玩家会花很多时间思考他们的线路，并分析他们比赛的技术细节，而忽略了心理因素。如果你参加了一场足球 “比赛”，你会计划热身、伸展、吃健康的食物、补充水分等等。比赛结束后，你可以放松一下，为下一场比赛做准备。你也应该同样规划你的扑克牌局。

建立扑克游戏的常规。一场牌局不是一个孤立的事件。就像我们足球的例子一样，实际的游戏过程只是一系列事件的一部分，这些事件包括准备、比赛和牌局后分析。大多数玩家会快速查看结果，然后转向其他游戏，但赛后分析几手关键牌或许会有所帮助。制定赛后例行程序还能提醒你一些重要的问题，从而提升你的游戏水平。

大多数扑克玩家根本不会考虑这部分游戏，但忽视它就像没有充分准备就参加体育运动一样错误。正确的赛后管理和准备工作正变得越来越重要。扑克游戏的技术水平在不断提高，因此你必须在每次比赛中都表现出色、扎实。由于准备不足而导致的失误会显著影响你的胜率，所以你必须为每一轮都做好充分的准备。如今的玩家技术失误越来越少，如果因为心态和准备不足而输掉比赛，想要挽回损失将非常困难。那么，你该如何为扑克赛前做好准备呢？

首先——这很简单——在开始游戏时保持精神饱满、专注放松。这能防止你迅速失控。当你疲惫不堪、心情不好或压力大时，爆冷输掉或其他不幸事件更有可能扰乱你的心理平衡，迫使你陷入 B 或 C 状态。

你可以在游戏前热身，保持良好的心态。让你的大脑有时间进入扑克状态，在开始游戏之前就开始思考扑克问题。这样，在困难时刻，你的反应会更快、更稳健。你可以用几种方法做到这一点。你可以观看扑克视频，回顾你的一些牌局，阅读一些在线牌局讨论，或者分析你上一局游戏。这可能有助于记住你想在游戏中解决的话题和漏洞。重新关注几个关键方面，可以帮助你为即将到来的牌桌挑战做好准备。准备的方法可能因玩家而异，但目的是一样的：你希望自己能够发挥出最佳水平。

除了心态之外，你还有更多根本性的问题需要考虑。首先，你必须弄清楚关于你的游戏牌局的实际情况。最重要的是，你能适应玩多少张牌桌，以及你能在这些牌桌上保持 A 状态多久？如果你没有明确的答案，那么你需要做一些工作。利用你的数据库来确定哪些参数有助于你发挥 A 状态，哪些参数会阻碍你发挥 A 状态。

以这种方式规划你的所有游戏牌局，让它几乎成为一个心理游戏计划。显然，你不应该连续玩 72 个小时，但许多玩家，尤其是在情绪失控的时候，会玩非常长的牌局，因为他们在追逐亏损——或者追逐一条鱼。通常情况下，你会输掉更多的钱，而且玩得可能比鱼还差。你的注意力在长时间的游戏牌局中必然会受到影响——这是人之常情。你在这种游戏中遇到的问题可能会延续到未来，因为巨额亏损和糟糕的游戏牌局是未来情绪失控的温床。

这就是为什么确定你的最佳游戏牌局长度至关重要。如果你喜欢在多张牌桌上苦战，或许一小时的牌局，中间休息五分钟，可能适合你，或者你也可以在最佳状态上玩两三个小时。使用你的追踪程序，追踪特定牌局长度后出现的重大亏损。对牌桌数量也应如此。如果你无法以 A 状态打牌，那么开很多牌桌是没有意义的。找出你能盈利的牌桌数量，并坚持这个数量，无论有多少好牌桌。如果你对常规的牌桌数量感到满意，可以随时尝试更多牌桌。评估这些决定如何影响你的胜率，并利用这些信息为自己谋利。

游戏时，排除所有干扰。不要看微信，不要和女朋友聊天，不要看视频，不要做任何分散你注意力的事情。你应该将 100% 的注意力放在当前牌局上。大多数玩家花费大量时间制定均衡的游戏计划，却忽略了一些基本要素，例如根据自己的心理承受能力选择合适的桌数，或者保持良好的心态。这些想法应该能帮助你提升牌局管理能力。仔细思考，你就能想出更多方法。

牌局管理为你的单次牌局提供了一个框架，而职业生涯管理则为你的整个牌局构建了一个心理结构。换句话说，就是你的职业生涯。重要的是要清楚自己在每次牌局中想要达到的目标。关键在于将所有事件、每次牌局和每手牌都视为一个宏伟计划的一部分。这个宏伟计划就是你的扑克生涯。

### 职业生涯管理

如果你的游戏计划涵盖了理论部分，那么职业管理就相当于游戏的心理部分。你需要清楚地了解自己为什么玩扑克以及想要实现的目标。首先，设定一些目标：你想要实现什么？目标有长期目标和短期目标。例如，你可能想达到一定的网站 VIP 级别，以一定的权益突破某个限制，等等。这种整体方法将提升你的扑克游戏水平和生活质量。

首先，它有助于减少情绪失控，因为你不会将单个回合或爆冷出局视为孤立事件，而是将其置于你的扑克生涯的大背景下看待。如果你从这个角度看待你的回合，即使是糟糕的回合也会显得不那么重要，因为它们只是漫长道路上的孤立事件。只要做好充分的准备，这条路会带你走向美好的未来。

其次，在扑克和生活之间找到平衡。生活并非只关乎扑克，但扑克服务于你的生活。作为一名职业扑克玩家，你玩扑克不仅仅是为了娱乐。这是你的工作，你应该表现得专业一些。即使你是业余爱好者，也应该管理好你的扑克活动，以便实现你的目标。无论是职业还是业余，你都应该设定目标，并尽力打好牌。

日常工作或生意与打扑克之间有着很大的区别。你通常去办公室上班，然后下班回家，把工作上的问题留在工作上，在家开始你的空闲时间。当然，你也可以在家考虑工作上的问题，但工作和家庭在物理上是分开的。扑克玩家的生活更加自由，因为你可以自由决定何时玩以及玩多少。你不受办公时间或截止日期的限制，而且通常在家工作。

扑克玩家的自由伴随着自我管理的责任。你独自对自己的行为、时间和资金管理负责，你必须保持良好的扑克生活平衡。一直像僵尸一样苦苦挣扎是没有意义的，因为从长远来看，这不仅会影响你的扑克游戏质量，还会毁掉你的生活质量。你必须建立一个框架，确保你有足够的空闲时间来放松身心，从游戏压力中恢复过来。接下来，我们将帮助你在日常生活中养成一定的规律和习惯，从而平衡地对待扑克。这样，你就能玩得更好，进步更快，并享受更多的空闲时间。

一份固定工作中的 “工作与家庭” 通常由固定的工作时间、休息时间和空闲时间组成。这样的安排对扑克玩家大有裨益。固定的作息时间可以构建你的生活和游戏。给自己充足的睡眠时间。合理安排你的时间，例如，早餐后进行一些牌局回顾和学习，并按照日常规律进行游戏。在固定的时间吃健康的饭菜。定期锻炼以保持身体健康。众所周知，即使在像国际象棋这样的游戏中，身体健康也非常重要。扑克游戏又何尝不是如此呢？一个身体健康的玩家在扑克游戏中可以集中注意力更长时间，也会更健康、更快乐。

接下来，我们将讨论扑克中最重要的技能之一：资金管理 (BRM)。

### 资金管理

除了上述软技能外，你还必须掌握一项硬技能，那就是管理你正在使用的资金。你的资金是扑克游戏的绝对核心。没有钱，你根本玩不下去。因此，你必须采取措施确保不输掉全部资金。由于扑克涉及运气和波动，没有人能保证你不会输掉全部资金。但你可以巧妙地管理风险和回报，从而最大限度地利用你的资金。

你必须找到一个平衡点，既能获得合理的利润，又不会承担过高的风险。如果你有 1000 欧元的资金，你可能会把所有资金都押在 10/20 的赌桌上，从而迅速晋级。回报很高，但风险令人难以接受，因为大多数时候你都会在一局牌结束时输掉资金。另一方面，用这样的资金玩 0.01/0.02 的牌局几乎不可能破产，但你也赢不了多少钱。

关键在于找到风险与回报之间的完美平衡，让你的资金增长最快，同时又不会面临破产的巨大风险。首先，你必须是一位赢家。资金管理充其量只能延缓输家必然会输的命运。因此，战胜游戏并努力提升水平至关重要。

假设你是一位赢家，那么没有其他收入的职业玩家和不依赖扑克资金的休闲玩家的情况会大不相同。

职业玩家必须更加谨慎地使用资金，因为没有资金，他们的职业生涯就结束了。这需要非常稳健的资金管理，范围从 100-500 个买入。如果你不依赖资金并且有固定收入，你可以使用更宽松的资金管理。尽管如此，最低限度应该是 30-50 个买入。虽然积极的资金管理可以让你更快地晋级，但如果你想最大限度地降低破产的风险，你必须遵循一些指导原则。

首先，激进的资金管理必须伴随着个人纪律。如果在当前级别中情况不顺，你必须迅速降级，并且必须避免损失任何资金。严谨的牌局管理也至关重要。其次，你必须确定自己在目标级别中是否仍然是一名盈利型玩家。你可能在较低级别中以更高的胜率赢更多钱，而不是继续升级并保持收支平衡。当你与更强大的对手对抗时，波动性会增加。它也会随着激进程度的增加而增加，而且由于 PLO 中的权益非常接近，你通常会冒险投入大量资金获得很小的利润。所以，如果你还没有准备好进行大量的平手牌较量，你可能需要先在当前级别中积累更多信心，然后再继续升级。

切勿在胆怯的资金下游戏。尤其是在 PLO 中，你必须承担激进游戏的风险，并谨慎行事。你不能总是等待拿到坚果牌。你必须确定自己在高级别和低级别中的表现是否一样好。合理的资金管理能让你更有信心，因为你知道你的资金没有风险。选择合适的级别也与舒适区有关。金钱并非对每个人都有绝对的价值。如果你不适应一手牌就能输赢一个月的房租，就不应该玩这么大的级别。如果你玩这个，你很可能会变得过于被动和谨慎——结果就是你会输掉所有的钱，因为你太努力地避免损失。还有另一个极端。如果你玩的级别很低，钱对你来说并不重要，你可能会只是赌博，而不是玩适合的扑克。

记住最重要的一点：如果不遵守资金管理规则，它将毫无用处。许多玩家在情绪失控时，或者在运气好的时候过于自信时，会违反他们的资金管理规则，将他们的全部资金置于风险之中。不要因为缺乏纪律而破产。通往山顶的道路并非陡峭险峻，你将拥有一段更安全、更轻松的旅程。
