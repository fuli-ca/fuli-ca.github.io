title: 第八章：牌面学校
---

到目前为止，大部分讨论都集中在翻牌前策略上。如果你不擅长在后续回合打牌，你刚刚学到的理论就毫无价值。

翻牌后的第一课从牌面结构开始，因为理解这些术语以及玩家范围如何与不同的牌面结构互动，会贯穿到快速职业手册的其余部分。随着你对 PLO 的熟练掌握，你会发现选择最佳策略主要取决于**牌面结构和对手的倾向**。

就像我跟学生们说的：一个不关注牌面结构的扑克玩家，就像一个不知道球点数的棒球投手。用通俗的话来说，**牌面结构决定一切**。虽然这节课很短，但集中注意力并做好笔记至关重要，因为我们会在接下来的章节中反复提到这些内容。

### 牌面结构的三大特征

每种牌面结构都有哪些定义性特征？其实很简单。每种牌面结构都由其同花性、连接性和等级来定义。

同花性表示牌面是彩虹牌、双色牌还是单调牌（单色牌）。大约 55% 的翻牌是双色牌，40% 是彩虹牌，5% 是单调牌。这就是为什么彩虹牌在 PLO 中被估值很低的原因。如果 60% 的牌面结构都是双色牌或单调牌，那就意味着我们拿着彩虹牌时会面临巨大的挑战。在多人底池中尤其如此，而且彩虹牌也无法拿到（或阻断）后门同花听牌。连接性指的是牌面上顺子和顺子听牌的数量，而等级则考虑了牌面大小。

### 牌面结构术语

为了确保我们理解一致，有必要回顾几个定义。第一个定义是**湿润度**，在扑克术语中，它指的是牌面能够形成顺子和同花的程度。我在进行大量指导的过程中发现，很多玩家对湿润度一词的使用过于宽泛，几乎到了把任何双色牌面都称为 “湿” 牌面的地步。你我都知道，湿润的牌面并非生来就一样，那么，我们如何描述两种不同的双色牌面（例如 J-5-3 和 K-J-8）之间湿润度的差异呢？

女士们，先生们，我很高兴向你们介绍**重度**。重度指的是牌面与翻牌前范围的互动程度，它的作用是填补我们刚才指出的语言空白。在接下来的几页中，你将能够更好地理解这些定义是如何相互关联的，但现在只需理解，当我们说一个牌面 “重” ，是指对手的翻牌前范围极可能完美契合这个牌面结构。更具体地说，重的牌面包含大牌，而轻的牌面包含小牌。

最后这两个术语很有用，因为它们提供了一种量化当前坚果牌脆弱性的方法。称一个牌面为**动态**意味着当前的坚果牌很可能在河牌圈发生变化。另一方面，称一个牌面为**静态**意味着当前的坚果牌不太可能在河牌圈发生变化。

### 牌面结构示例

我们现在要做的是通过描述几种不同类型牌面的不同动态，将牌面结构的定义变得生动形象。目标是让你熟悉哪些类型的范围对每种牌面击中力度最大、最频繁（以及最不频繁），以及哪些类型的牌会推动行动。

### 示例 1：重 / 干牌面（例如 A♦️K♣️6♥️、A♠️J♦️5♣️）

在 A♦️K♣️6♥️ 和 A♠️J♦️5♣️ 等重 / 干牌面中，筹码全押入底池通常是两手强成手牌碰撞，因为没有太多听牌可以半诈唬。例如，在 A♦️K♣️6♥️ 上，你唯一合理的听牌是内嵌包牌 (Q-J-T)，这是一个相当特殊的牌型组合。

重牌面的显著特征是百老汇牌的存在，而重牌面和轻牌面之间的主要区别之一在于暗三条的普遍性。大多数玩家会用所有大对子的组合加注、溜入或跟注，因此**当玩家在重 / 干牌面中全押时，通常是用顶两对或更好的牌**。你也可以想象一下，翻牌前的行动（以及玩家的范围）会如何变化。例如，这个牌面命中大多数 3-bet 范围，但命中后位较宽的开池范围的概率却远不及此。因此，在阅读牌面并考虑诈唬时，目标是牢记其他玩家的翻牌前范围以及你自己的范围。

如果你对这些术语感到困惑，或者不知道如何区分重牌、轻牌、湿牌或干牌，那么这里有另一种看待它的方式。看着牌面，只需问自己： “有多少牌对抗坚果牌有不错的权益？” 在干牌和重牌面的坚果牌通常拥有大量的权益。另一方面，在湿牌 / 重牌面的坚果牌通常对抗全押范围的权益较低。随着我们学习本章的剩余部分，你会对此有更深刻的理解。

### 示例 2：重 / 湿牌面（例如 J♦️T♠️6♦️，A♣️T♥️9♣️）

重 / 湿牌面与重 / 干牌面的动态不同，因为有很多牌有权益可以半诈唬。在重 / 干牌面的单次加注底池中，玩家通常会在翻牌圈持续下注，并试图以低成本打到摊牌，因为很少有中等牌力的牌愿意下注超过两条街，也没有多少牌可以用来半诈唬。在重 / 湿牌面，很少有牌没有权益，因此玩家更有可能半诈唬并下注以求保护，而不是试图以低成本打到摊牌。

**在 J♦️T♠️6♦️、A♣️T♥️9♣️ 和 K♥️J♦️8♥️ 等重 / 湿牌面，当你想争取全押时，听牌的坚果潜力至关重要。** 在这些翻牌面，用被压制的同花或顺子听牌全押会让你损失金钱。具体来说，像裸的非坚果包牌或没有再听牌的底三条这样的牌在这些翻牌面几乎没有价值，尤其是在多人底池中。

在 J♦️T♠️6♦️ 上，有很多牌对抗三条 J 有很好的权益。事实上，有些牌对抗裸顶三条是领先牌，比如 A♦️K♠️Q♥️9♦️。这就是为什么在多人底池中，中三条和裸顶两对在对抗那些愿意全押的典型范围时表现不佳的原因。这有助于解释为什么在听牌较多的公共牌面对子翻牌圈击中三条时，拥有后备性边牌或次要权益很重要。

### 示例 3：轻 / 干牌面（例如 9♦️4♠️2♣️、J♣️7♦️3♥️、5♦️4♥️2♠️）

虽然这些翻牌面和 A♦️K♣️6♥️ 都被认为是 “干燥” 翻牌面，但主要区别在于，在轻 / 干翻牌面击中暗三条的可能性大大降低，因为大多数玩家玩小口袋对子的频率远低于玩大口袋对子的频率。

同样，在不连接的翻牌面，玩家范围通常不会在翻牌圈击中两对，这就是为什么超对在这种翻牌面，尤其是在 SPR 较低的翻牌面，拥有巨大价值的原因。话虽如此，这肯定取决于你的超对的等级，因为像 9♦️4♠️2♣️ 翻牌上的 J-J-x-x 这样裸对子的牌，面对更大的超对时，状态非常糟糕。通常，**这些牌面非常适合拿着超对 3bet 底池的情况。对手不仅会频繁弃牌，而且你对抗那些全押范围的牌也表现得非常出色。**

由于这些公共牌结构中成手牌很少，所以不同牌型的权益差距很小。高牌、随机顺子听牌以及对子 + 边牌的组合非常丰富，这些牌型对抗超对和其他愿意在底池中投入资金的类似牌型时，权益在 35%-50% 之间。

### 示例 4：轻 / 湿牌面（例如 7♣️6♠️2♣️、9♦️6♠️4♦️、T♥️4♦️3♥️）

轻 / 湿牌面与轻 / 干牌面类似，翻牌前范围对这些翻牌的命中率不如重牌面，但可用的顺子和同花听牌数量差异很大。虽然拿到超强听牌的可能性降低，但大多数愿意投入底池的牌都包含各种组合听牌，例如对子 + 同花听牌。

与重 / 湿牌面类似，在全押筹码时，你的组合听牌的强弱程度至关重要。这是因为在轻 / 湿牌面拿到暗三条的概率非常低，因此，全押范围会更倾向于组合听牌。当筹码量没有飞快地押入底池时，**单次加注底池的行动需要在后续回合进行大量的缠打和操作，尤其是处于有利位置的玩家**，他们应该尝试展现出对手不太可能拥有的牌。我们将在第十章 “翻牌后战争” 中进一步讨论这些策略。

### 动态与静态

之前，我们区分了动态牌面和静态牌面。同样，动态牌面的当前坚果牌很可能在河牌圈发生变化，而静态牌面则相反。以下是一些不同程度的动态牌面和静态牌面的示例。

| 非常动态 | A♣️T♥️9♣️ | K♥️J♠️8♥️ | Q♦️T♠️6♦️ |
| --- | --- | --- | --- |
| 动态 | K♠️8♦️5♠️ | J♦️6♣️4♦️ | 7♣️6♥️2♣️ |
| 静态 | J♥️8♠️6♣️ | A♣️K♦️9♣️ | 7♥️7♠️5♥️ |
| 非常静态 | 8♥️5♥️2♥️ | K♦️6♣️6♠️ | A♥️K♣️6♦️ |

本课的重点**并非**是教你所有关于公共牌结构的知识。所有这些**是**让你养成注意不同范围如何与每种公共牌结构互动的习惯。我之前提到过，新手最大的漏洞之一就是过于害怕坚果牌，无论牌面结构如何。**换句话说，害怕坚果牌和意识到坚果牌之间存在很大差异。** 优秀的玩家知道坚果牌，但这并不意味着他们会被吓到而放弃最好的牌，因为他们了解不同的翻牌前范围与每种牌面结构是如何相互作用的。例如，当他们看到一个激进的玩家在 4-4-2r 牌面过牌 - 加注时，他们的第一反应不是弃牌，因为他们害怕三条四。他们的第一个想法是： “好吧，他这里三条牌不多，那么我该如何从他的空气牌中获得最大价值呢？”

**动态公共牌面上的利润来自于在与范围权益对决时占据主导地位，并在摊牌时用成手牌获得价值，而静态公共牌面上的利润更多地来自于诈唬，因为对手在静态公共牌面上击中的频率并不高。** 更简单地说，静态公共牌面上的战斗是由那些了解对手倾向并掌握翻牌击中频率的玩家赢得的。

### 对子牌面

对子牌面对于新手来说比较棘手。我经常会问： “当我被过牌 - 加注时我该怎么办？别人翻牌圈击中三条的频率是多少？当我翻牌圈击中三条时我应该采取什么样的策略？” 这些都是很好的问题，我们将在后续课程中讨论如何应对对子牌面。现在，我希望你了解对子牌面的两个主要特征：

- 哪张牌是对子
- 非对子的牌是比对子的牌大还是小。

回答我刚才提出的问题，**一手随机牌在翻牌圈击中三条或葫芦的概率约为 20%**。对子牌的等级以及它与对手翻牌前范围的匹配程度尤为重要，因为在 2-4-4 牌面中，翻牌圈击中三条或更好牌的可能性远低于在 K-J-J 牌面中。

同样，公共牌上对子的大小也会影响半诈唬的机会。例如，虽然在对子公共牌上听顺子和同花通常不是个好主意，但在某些情况下，高牌和同花听牌具有半诈唬的价值。假设你在 BTN 开池，一位筹码量 100bb 的稳健玩家在盲注位置进行了 3-bet。在像 5♦️5♣️6♦️ 这样的低公共牌上，持有同花听牌是有价值，因为他的 3-bet 范围不太可能击中这个公共牌，但你被感知的跟注 3-bet 范围肯定包含翻牌圈击中三条的牌。结合你的弃牌权益，在翻牌圈半诈唬加注是一个可行的选择，而且利润丰厚。

在对子高于非对子牌（7♦️7♣️4♠️）的公共牌面，玩法与相反的情况不同。主要区别在于更多的行动，因为幸运地击中三条的玩家会更轻松地快速打出他们的牌。毕竟，即使有人翻牌圈拿到葫芦，只要边牌还活着，他们仍然拥有约 40% 的权益，而且还有可能拿到最好的牌。此外，即使你的踢脚落后，只要边牌还活着，你的权益仍然合理。

另一方面，当对子比非对子牌（4♦️4♣️7♠️）低时，行动往往会显著放缓。尤其是在多人底池或深筹码底池中，持有三条的玩家不太愿意把钱投进底池，因为他们担心面对顶葫芦，他们几乎无法拿到任何补牌。大多数时候，持有裸三条的玩家会持续下注，并且在面对激进的玩家时保持被动，尤其是在公共牌上非对子牌与玩家翻牌前范围匹配时，比如 A♥️3♠️3♥️。

在这些公共牌上行动放缓并不意味着没有发挥创造力的空间。由于裸小三条必须考虑到另一位玩家持有顶葫芦，因此，当你持有阻断牌到坚果葫芦时，这是一个很好的机会来进行大诈唬。在尝试这样做之前，请确保你像对待其他阻断牌诈唬一样对待它。不要对着鱼或任何无法弃牌三条的玩家这样做，无论你下注的代价有多高。此外，确保筹码足够深，以便在必要时进行多次下注。最后，如果你的形象不好，就不要这样做。更多关于使用阻断牌的建议，请务必阅读第十一章。

### 第八章 概念小结

1. 约 55% 的翻牌是双色牌，约 40% 的翻牌是彩虹牌，约 5% 的翻牌是单调牌。这进一步削弱了彩虹牌在翻牌后的价值，这还没有考虑到彩虹牌无法在翻牌圈形成后门同花听牌。
2. 湿润度是指公共牌面产生顺子和同花听牌的可能性。重度是指公共牌面与玩家翻牌前范围中牌的互动程度。动态和静态指的是当前坚果牌的脆弱性。
3. 在动态公共牌面上，半诈唬和全押的决策由权益驱动。在静态公共牌面上，同样的决策则由频率和对手的倾向驱动。无论是在重 / 湿还是轻 / 湿公共牌面上，组合听牌的坚果性在全押时都至关重要，因为大多数全押范围都倾向于组合听牌。
4. 轻 / 干的公共牌面非常适合像持有超对进行 3-bet 底池这样的情况，因为对手不仅会对你持续下注弃牌很多，而且你对抗全押范围的牌也非常出色。
5. 害怕坚果牌和意识到坚果牌之间有很大区别。优秀的玩家总是意识到坚果牌，但这并不意味着他们会因为害怕而放弃最好的牌，因为他们了解不同的翻牌前范围如何与不同的公共牌结构互动。
