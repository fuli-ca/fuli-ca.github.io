# 第九章：持续下注

在之前的章节中，我提到所有扑克玩家都应该学习 PLO 的主要原因之一是它能让你在其他所有游戏中都表现得更好。从翻牌后的角度来说，持续下注的策略正是 PLO 和 NLHE 之间区别的开始。换句话说，要想成为 PLO 中的持续下注高手，你需要考虑 SPR、牌面结构和多街规划，而这些在德州扑克中是不需要的。

一个好的翻牌计划需要构建一个覆盖所有不同转牌和河牌可能性的范围。此外，我发现新手玩家在翻牌后最大的漏洞之一就是把翻牌当成一个真空状态。更具体地说，他们没有考虑翻牌圈的决策如何影响他们在转牌圈和河牌圈的盈利能力。

本章反复强调的一个要点是，思考我们是否持续下注会创造出哪些局面。我们应该如何根据牌面结构调整我们的下注尺寸？我们最常对哪些对手进行持续下注？在 3-bet 底池中，我们的策略会如何变化？这些都是我们将要回答的几个问题，所以事不宜迟，让我们开始吧。

### 主要问题

持续下注盈利的有效方法是，首先要诚实地评估被过牌 - 加注的可能性。这是一种有效的机制，不仅因为它迫使你盘点所有必要的翻牌后因素，还因为它迫使你权衡每一项因素的重要性。我讨论的主要因素包括：

- 对手数量
- 对手类型
- 筹码量
- 牌面结构
- 桌面形象

我们将在以下章节中逐一讨论这些因素，但例如，以上面列出的因素为指导，花点时间思考一下在四人底池中，在 K♠️J♣️8♠️ 牌面被过牌 - 加注的可能性与在单挑底池中，在 K♥️7♦️3♣️ 牌面被过牌 - 加注的可能性之间的差异。

### 牌面结构考量

作为一项总体策略，你在干 / 轻的牌面的持续下注频率应当高于湿 / 重的牌面。轻的牌面由低牌力组成，较少击中翻前范围。因此，干 / 轻的牌面受对手数量的影响较小，因为对手在此类牌面击中强成牌或听牌的概率降低，这意味着相比潮湿牌面，在多人底池中你可以更宽泛地使用持续下注。此外，多人底池中的持续下注也更容易获得对手的尊重。

干燥牌面也提供了良好的持续下注机会。玩家不仅在翻牌圈拿到顺子和同花听牌的次数更少，他们在转牌圈获得权益的次数也更少，这意味着你会从那些在翻牌圈玩 “不中就弃牌” 的玩家，或者那些在翻牌圈过宽的跟注并在转牌圈面对下注弃牌的玩家那里收获大量死钱。

湿 / 重的牌面的动态有所不同，因为对手可以继续对抗的牌更多。在单挑和多人底池中，你不仅更容易被过牌 - 加注，而且玩家还有更多牌可以跟注。基于上述在干牌面下注的原理，当你的权益较低时，应该减少在湿牌面下注的频率。与干燥牌面相比，对手更有可能要么完成听牌，要么在转牌圈获得额外的权益，从而有机会再次跟注或过牌 - 加注。

NLHE 玩家常犯的一个错误是过于频繁地用低权益的牌在湿牌面持续下注。这些牌面不适合在没有任何权益的情况下建立底池，因为猜测后面的街哪些牌可以下注、哪些牌可以停止下注会非常棘手。简而言之，德州扑克和 PLO 的区别之一是在 PLO 中你必须更频繁地投降。你可以诈唬成功的情况较少，因为玩家更容易拿着足够强的牌跟注。

### 持续下注尺寸

如果不讨论最佳下注尺寸技巧，就不可能写出一篇持续下注的章节。**最佳下注尺寸主要取决于牌面结构和 SPR，而真正的关键在于理解不同下注尺寸下，所有不同的转牌圈可能性将如何发展。** SPR 是多少？对手的跟注和加注范围有多大的弹性？以及可能的转牌圈结构有哪些？

我们先从牌面结构说起。通常情况下，在静态牌面下注较小，在动态牌面下注较大。原因很简单，与我们上一节的内容相关。玩家在静态牌面获得翻牌圈权益的几率较低，这意味着他们会更频繁地对持续下注弃牌。因此，下注较小是合理的，因为它能让你获得更好的诈唬价格。

另一方面，在动态牌面下注较大也是合理的，原因有几个。首先，动态牌面比静态牌面更容易击中范围，而且会改变牌面结构的转牌种类更多。你需要向对手听牌收取更高费用，同时当我们的半诈唬听牌击中时也能从对手跟注中获利。此外，下注越大，对手缠打或诈唬 - 加注的成本就越高，他们在动态牌面比静态牌面更容易这样做。在某些情况下，在动态牌面做较小的持续下注是合理的。这种情况发生在你持有超强听牌，并想鼓励具有反向潜在赔率的牌继续跟注时。在 8♦️4♣️2♣️ 牌上持有 A♣️8♣️8♥️7♥️ 就是这种情况的典型例子。

接下来，注意下注尺寸如何影响转牌圈和河牌圈的 SPR 非常重要。注意 SPR 可以让你在转牌圈利用河牌圈下注的威胁迫使对手弃牌，从而充分利用你的筹码。最后，如果计划用价值牌或强半诈唬牌在河牌圈全下，就必须相应地规划下注，以免在前面几条街下注过小而损失价值。NLHE 和 PLO 的另一个重要区别是，在 PLO 中，在河牌圈不能为了最大化价值而超池下注。

刚才我们概述了如何根据牌面结构和 SPR 来选择下注尺寸。你可能会想，持续下注满底池有意义吗？毕竟，这是一款底池限注游戏，那么是否存在下注最大底池有意义的情况？简而言之，是的，绝对有。除了在非常动态的牌面下注以对抗听牌，或者在 SPR 非常低的 4-bet 底池中下注这些显而易见的情况外，还有一些情况下注满底池是有意义的。

第一种情况是针对频繁过牌 - 加注和诈唬加注的玩家。加大下注是一种有利可图的调整，因为从博弈论的角度来看，如果他们选择再加注诈唬你，那么必须更频繁地奏效才能盈利。例如，假设我们在 100 的底池中下注 50，而一位玩家过牌 - 加注到我们下注的三倍，加注到 150。他冒着 150 的风险赢得 300，这意味着他需要一半的成功率才能盈利。另一方面，如果我们在 100 的底池中下注 100，而他过牌 - 加注到我们下注的三倍，那么他冒着 300 的风险赢得 500，这意味着必须有 60% 的成功率才能盈利。

此外，下注满底池通常被认为比持续下注标准的三分之二或一半底池更强。大约一年前，我感到很沮丧，因为几乎每次持续下注都会被跟注或加注。所以我调整了策略，对付那些我觉得会用宽范围跟注并诈唬加注的玩家，我会持续下注满池。结果，对手在面对我时变得更不敢轻举妄动了。这显然取决于对手是谁，因为有些玩家的行动对下注大小不敏感，所以对抗他们时，你最好坚持默认的建议。对抗那些不尊重你持续下注的玩家，持续下注满池绝对是一个不错的选择。

最后，持续下注满池比标准下注大小更能体现对手的范围。由于玩家通常害怕被听牌追上，他们没有平跟大牌的纪律性，这意味着如果你持续下注满池，他们会试图快打以保护自己的牌力，从而暴露自己的牌力。因此，当他们缠打你时，你就会知道他们的范围可能更倾向于听牌，这使得有利位置和不利位置的手牌阅读变得更容易。

### 对手数量

在之前的章节中，我们描述了对手数量如何影响我们的翻牌后决策。这会如何改变我们的持续下注策略？首要建议是在单挑底池中提高持续下注的比例。强牌组合的数量减少了，这意味着面对通常的对手时，被过牌 - 加注的可能性也会降低。

同样的道理，在多人底池中，你的持续下注频率必须显著降低。尤其是在湿润且重的公共牌面，很有可能有人在翻牌圈拿到坚果牌或足够强的听牌，从而将资金投入底池。如果你在三人底池中，在相关样本量下持续下注的比例约为 50%，那么你的策略就正确了。

**在三人底池中，翻牌前加注和翻牌圈下注之间的联系较少。** 但在四人或四人以上的底池中，这种联系几乎不存在，大多数时候，行动都集中在翻牌圈拿到坚果牌或接近坚果牌的玩家身上。在四人底池中，即使你拿着一手强牌，只要翻牌圈没击中就放弃底池，这完全是常态。

### 对手类型

在深入探讨本节之前，我想指出，根据经验，你必须在每个级别针对未知玩家制定一个默认的持续下注策略。我们当然希望掌握每位玩家的数千手牌和信息，但坦白说，这并不现实。

例如，一个良好的应对未知玩家的默认策略是在单挑底池中，几乎所有公共牌结构上都用你最强和最弱的牌持续下注，同时用中等强度的牌和听牌随后过牌。这很容易理解。否则，空气牌无法赢得底池，而用强牌构建底池当然是主要目标。用中等强度的牌，你可以通过随后过牌并实现权益来打一场双条街的游戏。

此外，翻牌后权益与翻牌前范围直接相关。这似乎显而易见，但我在扑克中最大的 “顿悟” 时刻之一发生在几年前我和我的 NLHE 教练的一次指导课上。这是一个简单的概念，我们在之前的章节中也多次提到过，但它的真正含义是**更紧的翻牌前范围更容易获得翻牌权益，而更宽的范围则更少获得翻牌权益**。面对更紧的翻牌前范围，如果你没有强牌，在湿 / 重的公共牌面持续下注，那简直是在烧钱。更紧的翻牌前范围主要由百老汇牌的单同花和双同花连牌组成，随着底池玩家数量的增加，这些因素的影响会被放大。

另一方面，更宽的翻牌前范围与更低的翻牌权益相关，这意味着他们没有足够的强牌来对抗持续下注。更紧的翻牌前范围更有可能包含更大、更连接的牌，而更松的范围则包含更小、更不连接的牌。此外，他们在后续回合获得权益的概率也更低，所以你不仅可以更多地持续下注，还可以更多地持续开枪。面对弱势被动的对手，你可以 100% 的频率持续下注，直到他们开始适应你。

目标是让你认识到每个玩家的翻牌前范围如何与牌面结构互动。这样，你可以从很少赢的公共牌面避免持续下注中节省资金，同时攻击那些错过其范围的公共牌。

### 延迟持续下注

知道何时随后过牌和知道何时持续下注同等重要。许多转型期的 NLHE 玩家过于频繁地持续下注，而没有考虑到在特定情况下随后过牌的好处。

最常见的随后过牌理由是对抗激进的过牌 - 加注者。另一种典型情况是：面对翻牌前紧的范围时，在湿 / 重的公共牌面缺乏弃牌率的情况下选择随后过牌，或多人底池中也是如此。如前所述，PLO 中玩家在翻牌圈实现权益的概率高于其他玩法，因此在权益渺茫的牌面选择放弃并无不妥。

最后，是否随后过牌很大程度上取决于你是否能在转牌圈继续下注，以及你有多少后门权益。随后过牌可以让你控制底池大小，并打一场两条街的游戏，这在面对激进的玩家时尤其有效。

虽然延迟持续下注的理由有很多，但要避免养成只要翻牌圈有后门权益就随后过牌的习惯。我见过很多玩家在翻牌圈随后过牌，而这些翻牌圈通常是标准的持续下注，因为他们认为自己有后门同花听牌、后门顺子听牌或其他形式的后门权益。要知道，在 PLO 中，你会拿到四张牌，这意味着你通常会有某种后门同花或顺子听牌。

**后门权益的存在并不会凌驾于牌局中的其他因素之上，例如对手的倾向或牌面结构。** 例如，假设你在 BTN 用 8♠️7♣️6♠️5♦️ 开池，BB 位置的一位松的玩家跟注。他在翻牌圈 K♠️3♥️2♣️ 过牌后，你拿着一个卡顺 + 后门同花听牌。面对这种类型的玩家，即使我们有很多好的转牌，下注也绝对比过牌好，因为考虑到牌面的干燥程度，他应该不会对我们进行频繁的过牌 - 加注。

面对激进的玩家，过牌是一种选择，但在 PLO 中，大多数情况下，给对手免费牌都是个坏主意。在这种情况下，我发现宁可激进，也不要被动。只要你有后门权益就过牌，很容易养成错误思考游戏的习惯。这会鼓励你依赖牌组来摆脱困境，而不是保持激进并赢取死钱，而这正是成功玩家的主要盈利来源。

简而言之，赢家总是在寻找下注的理由，而输家总是在寻找不下注的理由。在这个例子中，持续下注的另一个好处是，如果你不持续下注，那么在后续回合就很难有所作为，这意味着你放弃了拿下底池的机会。

### 3-bet 底池中持续下注

3-bet 底池中的大部分利润来自两个不同的方面。首先是成功持续下注后收回死钱。面对 3-bet 太轻的跟注看牌玩家，利润来自于跟注 3-bet，然后在翻牌圈持续下注时弃牌积累的死钱。我们希望能够轻的 3-bet 这些玩家，因为他们的弱范围经常错过翻牌，从而难以反击我们的持续下注。

第二种情况出现在对抗那些 3-bet 跟注范围偏紧的玩家时。此时利润主要来自在翻牌前进行纯粹价值向的 3-bet，然后在翻牌后压制他们的组合听牌。若采用第五章设定的范围，你在不利位置时持续下注的频率必定会比在有利位置时更高，因为你不利位置的 3-bet 范围本就强于有利位置。由于我们在有利位置会采用更宽的范围进行 3-bet，这种情况下看到全部三条街的公共牌往往更有利。而在不利位置 3-bet 底池时，我们更倾向于要么持续下注直接收池，要么快速下注获取价值。

最有效的持续下注牌面是干燥 / 静态的牌面。尤其是在干燥的 A 或 K 高翻牌面，你几乎可以 100% 地持续下注，因为它碾压你被感知的 3-bet 范围。对子牌面也适合持续下注，因为典型的范围在这些牌面类型的翻牌圈权益极低。

第二个利润来源是组合听牌的主导地位。了解哪些牌面应该下注 - 跟注、下注 - 弃牌和过牌 - 弃牌需要一些练习。如果你想提高自己的水平，我建议你查阅 PLO 快速统计，并在各种牌面和范围上进行一些模拟。

不过，我建议你拿着空气牌在那些又重又湿的公共牌放弃。在面对直接对手时，在 A 高牌的湿公共牌持续下注还是没问题的，这取决于牌面的连接性。但通常情况下，在 T-9-8 这样的公共牌上持续下注，如果你没有击中一点翻牌，那简直就是在烧钱。你可能会想，这肯定很容易被剥削利用，因为我们的打法完全不平衡，对吧？其实，解决办法就是，当你翻牌击中强牌时，在这些公共牌面过牌 - 加注，这样对手就没办法总是在你过牌时下注了。很多玩家在 T-9-8 这样的公共牌上，在 3-bet 的底池里，面对过牌会选择轻的下注偷池，因为看起来你用 A-A 3-bet，然后放弃底池，所以当你翻牌拿到强牌时，很容易让对手轻的跟注全押。

### 第九章 概念小结

1. 牌面结构在决定最佳翻牌后策略方面起着关键作用，可以说是你持续下注前需要考虑的最重要的因素。干燥和轻的牌面是静态的牌面，对手可以继续玩的牌相对较少。因此，我们应该更频繁地持续下注，并建立有利可图的持续下注局面。湿润和重的牌面是动态的，对手可以继续玩的牌更多，因此我们应该降低持续下注的频率，尤其是在我们错过牌面或是多人底池游戏时。
2. 在 3-bet 底池中，大部分利润来自于通过持续下注捡到死钱以及主导组合听牌。通常，你应该在不利位置时增加持续下注，在有利位置时减少持续下注。在干燥 / 静态的牌面最常进行持续下注，尤其是在面对那些在 A 或 K 高牌面玩法直接的对手时。
3. 最佳持续下注尺寸取决于牌面结构和 SPR。在静态牌面，我们应该持续下注更小但更频繁，而在动态牌面，我们应该持续下注更大。考虑到 SPR，多街规划是利用转牌圈杠杆来剥削对手的关键。此外，如果你想全押筹码获得价值，或者留下足够的资金进行有效的诈唬，你需要将河牌圈的 SPR 设置为 1 或更低。一般来说，在不利位置时，你应该持续下注更大，在有利位置时，你应该持续下注更小。
4. 持续下注满底池对频繁过牌 - 加注和诈唬加注的对手很有用。这降低了他们诈唬的赔率，而且会被认为是一手强牌。它也对那些会明确自己范围的对手有用，他们通常会用范围顶端的牌加注，然后用中等强度的牌平跟。
5. 对手的数量显然会影响我们的持续下注决策。在单挑底池中，我们通常应该以更高的频率持续下注，大约 65%。记住要根据牌局中的具体对手进行调整。在三人底池中，我们平均仍然应该在大约一半的时间里持续下注；在湿润 / 重的公共牌面，如果我们错过翻牌，那么持续下注的次数要少得多。在有四个或更多对手的底池中，除了可能最干燥的公共牌面之外，主动权并不重要。你应该坚持追求价值。记住，在多人底池中，你被感知到的持续下注范围会被认为相当强。
6. 翻牌后权益取决于翻牌前的牌型选择。紧手玩家往往在翻牌圈获得更多权益，而松手玩家则在翻牌圈获得更少权益。这一点，再加上公共牌面结构，应该可以帮助你决定是否对对手持续下注并后面的街继续下注。
7. 在某些情况下，随后过牌是更好的选择。首先，当我们放弃这手牌时。这通常发生在湿润公共牌面，当我们完全错过翻牌或处于多人底池中时，因此我们的弃牌权益会显著降低。其次，用我们平滑的中等强度牌随后过牌来对抗频繁的过牌 - 加注者，由于我们所有的后门权益和良好的可玩性，我们可以在两条街的游戏中获利。
