## 介绍

假设你做为翻牌前进攻方在单挑底池翻牌持有你的底端范围。牌面相对干燥，你的 HUD 显示对手对持续下注的平均弃牌率为 55%。现在，如果你能知道自己用 3/4 底池大小的持续下注是否自动盈利，那岂不是很棒？

另一种情况是你在河牌圈，你的大听牌没中。对手下注了 1/2 底池。再说一次，如果能知道平均需要多少弃牌权益才能让一个底池大小的诈唬加注有利可图，那该有多好啊！

这就是在决策中考虑弃牌权益的重要性所在。请记住，不仅仅是 “F2Cb” 数据决定了你的弃牌权益。还有其他更重要的因素，例如牌面结构、你的下注大小、底池中的玩家数量以及剩余的 SPR。你的弃牌权益会增加：

- “F2Cb” 数据越大
- 牌面结构越干燥
- 你下注越大
- 底池中的玩家越少
- SPR 越大

只有结合所有这些因素，你才能全面估算出你在特定场合的实际弃牌权益。网络上有一些不错的免费弃牌权益计算器，只需谷歌搜索一下即可。使用这些计算器，你应该能够完成以下练习。我建议你尝试不同的大小来感受弃牌权益。

对于这些练习中的计算，您可以使用权益计算器。或者，直接使用我们之前已经讨论过的数学方法。

## 测验

1. 我们需要多少弃牌权益才能盈利：a) 底池大小的诈唬，b) 3/4 底池大小的诈唬，c) 2/3 底池大小的诈唬，d) 1/2 底池大小的诈唬？
2. 底池为 $20，对手对 c-bet 的弃牌率为 45%。
在这种情况下，我们最多可以进行多少 c-bet 才能自动盈利（仅考虑对手对 c-bet 的弃牌率）？
3. 底池为 $100，对手下注 $70，Hero 加注到 a) $310，b) $225，c) $155
Hero 在这种情况下需要多少弃牌权益才能使纯诈唬加注盈利？
4. 我们在这里观察到的一般模式是什么（考虑底池大小的诈唬加注、3/4 底池大小的诈唬加注和 1/2 底池大小的诈唬加注的弃牌权益）？

## 解答

1. **我们需要多少弃牌权益才能盈利：a) 底池大小的诈唬，b) 3/4 底池大小的诈唬，c) 2/3 底池大小的诈唬，d) 1/2 底池大小的诈唬？**
    1. 例如：在 $100 的底池中下注 $100：((100 / (100)+100)) = 50% 弃牌权益
    2. 例如：在 $100 的底池中下注 $75：((75 / (100)+75)) = 43% 弃牌权益
    3. 例如：在 $100 底池中下注 $66：((66 / (100)+66)) = 40% 弃牌权益
    4. 例如，在 $100 底池中下注 $50：((50 / (100)+50)) = 33% 弃牌权益
2. **底池为 $20，对手对 c-bet 的弃牌率为 45%。
在这种情况下，我们最多可以进行多少 c-bet 才能自动盈利（仅考虑对手对 c-bet 的弃牌率）？**
    
    这与练习 1 的方法类似。这次，只有变量发生了变化：$x 在 $20 底池下注：x/(x + 20) ≥ 0.45
    其余部分只是对不等式进行变换：0.45 × (x+20) ≥ x 0.45x + 9 ≥ x 9 ≥ 0.55x
    x ≤ 9/0.55
    x ≤ $16.36
    
    因此，我们可以下注最多 $16.36，从而在这个场合自动盈利。
    
3. **底池为 $100，对手下注 $70，Hero 加注到 a) $310，b) $225，c) $155
Hero 在这种情况下需要多少弃牌权益才能使纯诈唬加注盈利？**
    1. 310 / ((100+70)+310) = 64.5%
    2. 225 / ((100+70)+225) = 57%
    3. 155 / ((100+70)+155) = 48%
4. **我们在这里观察到的一般模式是什么（考虑底池大小的诈唬加注、3/4 底池大小的诈唬加注和 1/2 底池大小的诈唬加注的弃牌权益）？**
    
    对手下注越少，我们加注所需的弃牌权益就越低。
    对于底池大小的诈唬加注，我们需要大约 65% 的弃牌权益。
    对于 3/4 底池大小的诈唬加注，我们需要大约 57% 的弃牌权益。
    对于 1/2 底池大小的诈唬加注，我们需要大约 <50% 的弃牌权益。
    

## 练习

如果你还没有练习过，那么你应该找些时间，使用你选择的弃牌权益计算器，尝试不同的弃牌权益。当你感到自信时，在考虑用你的底牌范围 c-bet 时，也应该考虑弃牌权益。请记住，即使使用你的绝对底牌范围，你在下注时也几乎总是至少拥有一定的权益（俗话说：“你下注，你赢”）。尝试找出你的下注大小所需的弃牌权益。如果你拥有该弃牌权益，请考虑牌面结构和对手的 c-bet 弃牌数据。
