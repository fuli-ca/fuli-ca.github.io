# 第十九天 - 高级底池赔率

在本节中，我们将讨论玩 PLO 所需的最重要的数学概念。它之所以如此重要，是因为如果你能练习并掌握这里的方法，你应该能够在任何情况下几乎立即（10 秒内）计算出底池赔率。如果你非常擅长心算，那么你的速度会更快。

## 介绍

我们已经完成了根据补牌估算权益的练习。最后一步是估算我们获得的底池赔率，并比较这两个值。以前，我的扑克教练告诉我，玩牌时要随身携带一个计算器，以便计算底池赔率。因为我不想在玩牌时使用计算器（即使是在线上，那也太书呆子气了），所以我成功地开发出了自己的 “秘密公式”，用于在游戏中执行估算。估算底池赔率非常简单，只需记住以下三个步骤：

1. 计算 “$pot”
2. 将跟注 / 加注额度与 “$pot” 关联起来
3. 使用我们的 SPR 表格检查结果

如你所见，在前两个步骤中，你需要知道 “$pot” 变量。那么，让我们看看如何计算这个神奇的数字：$pot = 下注前底池金额 + [Hero 下注额（或加注额）x2] + [跟注金额]

让我们来看看等式的右边。

下注前底池金额：任何下注发生之前底池中的金额。
Hero 下注额（或加注额）x 2：加上 Hero 最后下注（或加注）金额的两倍。
如果 Hero 没有下注或加注，则此值为 0。
跟注者：如果底池中有跟注者（“死钱”），则只需将每次跟注的金额相加。此变量仅在多人底池中计算。在单挑底池中，这个值始终为 0。

就是这样！如果你还没完全理解，不用担心，我们会提供大量示例供你练习。以下示例将分别使用以下变量，以便你理解该方法：

底池：$100

Hero 下注 $50，BB 跟注 $50，CO 加注到 $250，Hero 呢？在跟注之前（剩余金额 > $200），我们需要先了解一下步骤：

**计算 $pot**

为此，我们需要将上面列出的三个变量相加。Hero 下注前的底池金额为 $100。Hero 本身的下注金额为 $50。公式要求我们将这个下注金额乘以 2，因此 $50 × 2 = $100。我们有一个跟注者（BB），他跟注了 $50。因此，我们有：1）$pot = $100 + $100 + $50 = $250

**将你的跟注或加注金额与 $pot 关联起来**

第二步是将我们要跟注或加注的金额与 $pot 关联起来。在本例中，我们想要跟注，因此我们必须将跟注金额（即 $200）与 $pot 关联起来：$200 赢 $250 = 4/5（即略大于3/4）

**使用我们的 SPR 表检查结果**

最后一步是解密结果 “3/4”。这时我们需要用到上一节创建的 SPR 表。3/4 是 SPR 表的 “分数 (SPR)”。SPR 表告诉我们，3/4 意味着我们需要 30% 的权益。由于 200/250 (0.8) 大于 3/4 (0.75)，我们知道我们需要的权益略高于 30%。

如果你数学不好，只需判断 200 小于（1 倍）250，即可得出我们需要 <33% 的权益才能全押。使用这种方法，我们虽然偏离了数学上的正确答案，但你仍然应该得到一个可用的结果。顺便说一下，在这种情况下，数学上正确的答案是 200 / ((100+50+50+250)+200) = 30.8%

就是这样！现在该练习了。

## 解答

1. **底池：$35**
Hero 下注 $35，对手加注到 $126，Hero 全押？（后手 $62）
    1. **Hero 需要多少权益才能跟注？**
    i) $62 赢 $70（底池）+ $97（对手全押）* = $62 赢 $167
      167/62 = 2.7 : 1（3.7 次中 1 次）= 1/3.7 = 27%
    ii) 62 / ((35+35+97) + 62) = 27%
        
        * 注意，我们必须考虑有效筹码（Hero 跟注 $62）
        
    2. **我们如何估算？**
        1. $pot = $35（下注前底池金额）+ $70（Hero 下注金额 × 2）= $105
        2. 62/105 ≤ 2/3
        3. ≤ 2/3 ≤ 28.4% 权益
2. **底池：$15**
SB 下注 $15，Hero？（后手 $47）
    1. **如果 Hero 想全押加注，他需要多少权益？（不包括弃牌权益）**
        1. $47 赢 $15（底池金额）+ $47（SB 下注金额 + Hero 跟注全押）= $47 赢 $62
        62/47 = 1.3 : 1（2.3 次赢 1 次）= 1/2.3 = 43%
        2. 47 / ((15+15)+47+32) = 43%
    2. **我们如何估算？**
        1. $pot = $15（下注前底池金额）* = $15
        2. 47/15 > 3
        3. > 3 > 42.5% 权益
        
        * 注意，此处 Hero 的加注金额为 0，因为 Hero 尚未加注
        
3. **底池：$15**
Hero 下注 $15，对手加注 $30，Hero？ （后手 $47）
    1. 如果 Hero 想要全押加注，他需要多少胜率？（不考虑 FE）
        1. $32 赢 $30（底池金额）+ $47（SB 全押金额）= $32 赢 $77
        77/32 = 2.4 : 1（3.4 次赢 1 次）= 1/3.4 = 29%
        2. 32 / ((15+15+30)+32+17) = 29%
    2. **我们如何估算？**
        1. $pot = $15（下注前底池）+ $30（Hero 下注 × 2）= $45
        2. 32/45 > 2/3
        3. 2/3 > 28.4% 权益
4. **底池：$10**
SB 下注 $5，BB 跟注 $5，UTG 跟注 $5，Hero 加注到 $15，SB 跟注 $10，BB 加注到 $50，UTG 跟注 $45，Hero 呢？（后手 $35）
    1. Hero 需要多少权益才能跟注？
        1. $35 赢 $10（底池）+ $5 （SB 下注）+ $5（BB 跟注）+ $5（UTG 跟注）+ $15（Hero 加注）+ $10（SB 跟注）+ $45（BB 加注）+ $45（UTG 跟注）= $35 赢 $140 = 4:1（5 次赢 1 次）= 1/5 = 20%
        ii) 35 / ((10+5+5+5+15+10+45+45)+35) = 20%
    2. **我们如何估算？**
        1. $pot = $10（下注前底池）+ $15*2（Hero 加注 × 2）+ $5 + $5 + $10 + $45（跟注者）= $105
        2. 35/105 = 1/3
        3. 1/3 = 20%
    
    通过这些练习，你现在可能也明白了为什么我们的 SPR 表中既需要像 1/3 这样的小 SPR，也需要像 3 这样的大 SPR。
