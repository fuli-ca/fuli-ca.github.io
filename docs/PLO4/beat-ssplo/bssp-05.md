# 第五天 - 翻牌后基本打法

我们学习 PLO 的第一周已经进行到一半了。由于我们想要从头到尾准备如何玩一手牌，所以在讨论完翻牌前打法之后，是时候讨论最重要（也是最合乎逻辑）的主题了。那就是翻牌后打法的基础知识。本课主要讲解翻牌的策略。

## 介绍

你可以将这个主题与一步一步的翻牌前计划进行比较，以便用基本组成部分来解释。然而，与翻牌前玩法相反，用几个步骤来描述你在翻牌应该做什么要困难得多。由于翻牌会发出三张公共牌，因此需要考虑的打法变量和排列组合会成倍增加。其中最大的变量之一是翻牌的公共牌。你必须思考翻牌对你和对手的牌可能产生的不同影响。此外，不同的公共牌面也会影响你的翻牌权益。随着需要考虑的变量越来越多，需要评估的数据也越来越多。因此，五步计划并不适用于翻牌打法。毕竟，翻牌后的打法才是真正区分优秀扑克玩家和普通玩家的关键。在翻牌前确定牌型范围，并根据对手的翻牌后动作运用读牌技巧，这些概念至关重要。所有仍在底池的玩家都在翻牌前跟注或加注，因此考虑到玩家类型，我们至少可以掌握一些关于他们牌型范围的信息。

并非所有翻牌都与翻牌前的考量完全无关。我们仍然需要重新思考一些我们已经讨论过的翻牌前要点，因为现在我们到了翻牌，我们对翻牌前发生的所有事情都有了精确的信息。现在，我希望你完成练习，我将在解答部分解释每个细节。如果你努力练习并思考你已经学到的知识，我相信你一定会找到很多有用的要点。

## 测验

在做决定之前，找出你在翻牌应该考虑的最重要的要点。将它们按重要性从高到低排序，以便你能将它们应用到每一手牌中。并尝试证明你的观点的合理性。

## 解答

与翻牌前不同，翻牌更难找到一个准确的顺序，因为有些要点会分化成额外的要点。而且，在一手牌中至关重要的要点在下一手牌中可能完全不重要。奥马哈的翻牌就像雪花一样 - 没有两片是相同的！因此，在翻牌使用固定的要点顺序并不像翻牌前那样有效。尽管如此，我仍然想向你展示我个人认为合理的顺序的通用方法。

1. **到目前为止发生了什么？**
    
    我们必须知道谁是翻牌前进攻者，以及翻牌前发生的所有事情。如果我们在有利位置，那么我们已经掌握了之前玩家在翻牌实际行动的额外信息。例如，知道 UTG 翻牌前进攻玩家过牌，并且另一位玩家也对我们过牌非常重要。
    
2. **底池里玩家数量**
    
    底池中剩余的玩家数量会影响我们的决策。通常情况下，玩家越多，我们下注的弃牌权益就越低。玩家数量也会对我们的最佳打法产生重大影响。当我们在翻牌单挑时，公共牌是 7♥️-7♣️-2♦️，诈唬性过牌 - 加注通常会成功，因为当对手选择持续下注时，这是一个非常适合我们诈唬的公共牌。这是因为他在这个公共牌上击中好牌的机会非常小。但是，如果我们是 5 人底池，那么很可能有人拿着 7。所以在这里，诈唬性过牌 - 加注是一种无利可图的打法。
    
3. **筹码底池比 (SPR)**
    
    SPR 描述的是玩家剩余筹码量与底池的比率。谈到 SPR，我们总是要考虑有效筹码量，即仍在底池中且筹码量最少的玩家的筹码量。如果该玩家的筹码量太小，以致于变得无关紧要（就弃牌权益而言），那么你应该考虑下一个最小的筹码的玩家。
    
    SPR 在未来课程中会是一个重要的主题（它甚至有专门的课程）。昨天我们在翻牌前考虑对手的筹码量时已经讨论过 SPR。在任何下注轮次，尤其是在翻牌圈之前，都要养成在做任何决定时检查 SPR 的习惯。SPR 至关重要，因为它决定了：
    
    - 我们需要下注多少才能在转牌圈 / 河牌圈全押（第 17 天）。
    - 如果其他人全押，我们需要多少权益跟注（第 18 天）。
    - 我们得到的潜在赔率是多少（第 25 天）。
    - 我们的弃牌权益是多少（一个简单的经验法则是，SPR 越大，我们的弃牌权益就越高）。
    - 在后续回合中，我们有多少行动空间（例如，诈唬）和缠打。
    - 正如你所见，从现在开始，SPR 将成为我们的一个重要考虑因素。
4. **公共牌**
    
    说到公共牌，翻牌有 22,100 种可能的牌型组合。话虽如此，我们基本上可以将所有这些组合分为四种公共牌面类型：
    
    - 湿公共牌面 (5♥️-8♥️-10♣️)
    - 半湿公共牌面 (5♥️-7♥️-Q♠️)
    - 干公共牌面 (K♥️-7♠️-3♦️)
    - 锁定公共牌面 (4♥️-5♠️-8♦️、A♥️-A♠️-K♦️、2♥️-7♥️-K♥️)
    
    湿公共牌面是指当前的坚果牌在下一轮最有可能改变的公共牌面。在上面的例子中，任何 4、5、6、7、8、9、J、Q 以及任何红桃（共 28 张牌）都可以组成顺子或同花。任何一张转牌都会改变翻牌圈中当前绝对坚果牌的最佳牌型（一对 10）。与湿公共牌面相反的是锁定公共牌面，任何未来的牌都不太可能改变当前的坚果牌。你可以概括地说，牌面越干燥，你的翻牌权益就越高。
    
    步骤 7 和 8 将帮助我们更深入地了解如何应对这些牌面。我们还将在第三周学习更多关于此主题的高级思考。
    
5. **位置**
我们是否有绝对位置？我们最后行动的位置在哪里？如果没有，那么在多人底池的情况下，我们和翻牌前进攻方的相对位置是什么？
6. **玩家类型 / 数据**
再次，我们应该对翻牌遇到的玩家进行快速分类（基于第三天的 6 个玩家类别）。如果我们曾在这张牌桌或之前的游戏中与这些对手交手过，那么我们应该掌握这些信息。如果我们仍然不确定，那么我们应该参考一些数据来做出最佳评估。
7. **我们的手牌**
同样，我们自己的手牌几乎是事后才考虑的。除非你翻牌拿到绝对的坚果牌，否则除了我们实际的手牌之外，还有很多其他因素需要考虑。在评估了翻牌后所有其他牌桌动态后，以下是一些关于我们自己手牌的需要考虑的事项：
    - 我们的手牌与公共牌的匹配程度如何（就对子或听牌而言）？
    - 我们阻断了哪些成手牌 / 听牌？
    - 我们压制了哪些成手牌 / 听牌？
    - 我们的手牌在转牌和河牌玩起来有多容易？我们能否避免在未来几条街做出艰难的决定？多少张转牌可以改善我们的手牌？
    - 我们是否有后门（即连续有帮助的牌）听牌？
8. **感知范围 / 权益 vs. 范围**
根据玩家类型和翻牌前的行动，我们应该能够相当准确地判断对手可能持有的手牌范围。最后要考虑的是，根据这个预期范围，估算我们的权益（即赢得或分享底池的概率）。
    
    正如你所见，有很多因素需要考虑 - 甚至还有一些我们尚未详细讨论的因素。但别担心 - 我们将一如既往地循序渐进地学习。今天应该会让你大致了解翻牌看到三张牌时需要考虑的因素。通过自律和练习，这些翻牌后的思考将成为你的第二天性。
    

## 练习

1. 将本课的 8 个要点抄写在纸上并打印出来。能够用我们今天讨论的理由来解释这些要点。正如我之前提到的，目前，没有必要把所有东西都背下来。但你至少应该能够大致理解每个翻牌要点对整手牌的影响。我们将在以后的课程中再次讨论这些要点。
2. 当你准备好时，进行一局游戏，并记录每手牌中翻牌后评估的要点。哪些要点在拖入底池时有效（就决策而言）？哪些要点是你忽略或误用，导致你输掉底池的？游戏结束时，查看你标记好的笔记，并尝试专注于你感到自信的要点和需要更多练习的要点。
