# 第十七天 - 两对打法

在 PLO 中，我们还没有讨论过一类牌型 - 两对。今天，我们来讨论一下这种有趣的牌型。

## 介绍

我的学生几乎都容易打错两对，损失不必要的资金。很多学生倾向于打每一手两对，因为他们认为翻牌圈击中暗三条的概率相对较高（确切地说，23.4%，即每 4.25 手牌中一次）。虽然这听起来很不错，但我们必须考虑到，我们的两对在翻牌没有击中暗三条的概率高达 76.6%。如果没有翻牌击中暗三条（并且没有潜在的听牌），你的两对在翻牌后几乎总是落后。此外，击中小暗三条（2-2-2 到 6-6-6）在德州扑克中可能很棒，但在 PLO 中，我们真的不应该太兴奋。想象一下，你在 UTG 开池加注 Q-Q-2-2r，3 位玩家跟注。翻牌是 10♠️-9♠️-2♦️。我们在翻牌下注，结果被我们标记为松散被动的玩家加注。我们愿意在这里弃牌吗？我觉得不会。另一种常见的情况是，我们在翻牌用暗三条 2 下注，然后在两个场合被跟注。转牌是 7♥️。即使没有拿到坚果顺子听牌（Q-J），我们真的愿意在这么大的底池下注吗？因为一旦有人加注，你就可能弃牌了。如果转牌真的来了一张 K、一张 8 或一张同花呢？即使是公共牌是 10 的对子也可能有风险。德州扑克不像德州扑克那样，暗三条非常强，很少输给暗三条。而在像 PLO 这样波动性更大的游戏中，由于可能更大的暗三条，并且有完成的听牌，我们可能不得不考虑采取更弱的策略。

希望你现在明白了过度玩弱两对牌的严重问题。暗三条输掉的可能性比德州扑克高得多。如果公共牌上的对子牌比你的暗三条大，导致别人拿到了更大的葫芦，即使是葫芦也可能造成灾难性的后果。一般来说，我们很少有机会玩这些牌，并在第五街下注，或者冒着全部筹码的风险对抗对手的攻击。因此，我们必须更加谨慎地选择我们想要玩的两对，并且更加谨慎地玩它们。

既然我们知道不应该防守甚至不要开池加注所有两对，那么现在是时候定义一个可玩的范围了。我们必须区分有利位置和不利位置，并且意识到我们的有利位置范围应该比不利位置范围更宽。我们必须确定一个可玩的牌范围，该范围必须满足以下条件：

- 至少包含一个非常高的对子。
- 包含两个相连的中等对子
- 包含两个相连的双同花低对子

## 测验

1. 我们应该用哪些两对来防守单个对手开池加注？将它们分为顶端、中端、底端范围。
2. 我们应该跟注哪些范围，以及哪些范围我们应该在 a）有利位置 b）不利位置，3-bet？为什么？
3. 当已有 2 名或更多玩家加入底池（且没有 3-bet 时），我们该如何处理我们的范围？

## 解答

1. **我们应该用哪些两对来防守单个对手开池加注？将它们分为顶端、中端、底端范围。**
    
    在继续之前，对我们的领对进行分类非常重要。顶端范围是所有高两对。中端范围代表高低两对的组合。最后，底端范围是低中连通两对。
    
    顶端范围：A-A-P-Pss、Q-Q-P-Pds+、J-J-8-8ss+、10-10-6-6ds+
    
    中端范围：Q-Q-P-Pss、J-J-4-4ss、J-J-8-8rb
    
    底端范围：8-8-5-5ss+、6-6-4-4ss+、4-4-3-3ds+
    
2. **我们应该跟注哪些范围，以及哪些范围我们应该在 a）有利位置 b）不利位置，3-bet？为什么？**
    
    现在我们已经确定了有利位置和不利位置两对可防守的范围，我们应该确定哪些应该跟注，哪些应该 3-bet。同样，在防守时，了解我们为什么更倾向于一个选项而不是另一个选项非常重要。
    
    表 2 应该能给你一个基本的思路。
    
    |  | 有利位置 | 不利位置 |  |
    | --- | --- | --- | --- |
    |  |  | SB | BB |
    | 顶端范围 | 3-bet | 3-bet | 3-bet |
    | 中端范围 | 跟注 | 3-bet | 跟注 |
    | 底端范围 | 3-bet | 3-bet | 跟注 |
    
    表2：基于位置的两对行动矩阵
    
    **a) 有利位置**
    
    - 我们用中端范围跟注，用顶端和底端范围 3-bet。
    
    我们用中端范围跟注的原因之一是，如果我们击中较弱的暗三条，我们更有可能用有利位置击败对手，并控制底池大小。如果是多人底池，也不会那么麻烦，因为我们总是有高对和同花听牌，可以在翻牌（非全押）的情况下压制更多的玩家。我们在有利位置用其余的范围 3-bet，因为我们希望用底端范围在翻牌前和翻牌后创造弃牌权益，同时用顶端范围榨取价值。
    
    **b) 不利位置**
    
    - 我们在 SB 用所有三个范围 3-bet。
    - 我们在 BB 用我们的顶端范围 3-bet，并跟注我们的中端范围和店端范围。
    
    不利位置（SB）：我们不能在 SB 跟注我们的中端范围，因为我们无法确定能单挑，因为 BB 仍然可能跟注，而且我们翻牌后不是有利位置。因此，我们无法很好地控制底池。
    
    不利位置（BB）：即使我们在 BB 不是有利位置，我们仍然可以跟注我们的中端范围和底端范围，因为我们在翻牌圈总是会单挑。我们用我们的顶端范围 3-bet，目的是榨取价值。此外，当我们在 3-bet 底池中击中暗三条时，我们可能会获得更多回报，因为 SPR 已经较小，因此对手会更宽地弃牌。最后一个原因是，当我们 3-bet 时，我们在翻牌圈更多是单挑，因此我们被更高暗三条压制的风险更低。
    
3. **当已有 2 名或更多玩家加入底池（且没有 3-bet 时），我们该如何处理我们的范围？**
    - 我们挤压我们的顶端范围
    - 我们跟注我们的中端范围
    - 我们跟注我们的底端范围
    
    我们用我们的顶端范围加注以获得价值，用我们的中端范围跟注以获得顶暗三条（有额外的同花听牌潜力），用我们的底端范围跟注（只为了我们获得的好赔率），并在击中暗三条时保持高度谨慎。
    

## 练习

1. 在你的不利位置防守图表的 “不利位置跟注” 部分中，用我们应该在 BB 跟注的牌填入空白处。同时，将那些你在 BB 跟注，但在 SB 仍然会 3-bet 的牌从你的不利位置 3-bet 部分中排除，方法是用括号将这些范围括起来。
2. 在你加入牌桌之前，准备一副牌。拿出牌堆最上面的两张牌，然后翻阅剩余的牌，直到你把这两张牌都凑成对子（这样你的牌型会更有变化）。将每对牌单独放在一个牌堆，例如 2-2 或 K-K，并移除其他两张相同点数的牌。重复此操作，直到你将所有对牌单独放在一个牌堆（13 张对牌堆，其余 26 张牌弃掉）。现在，随机（有无数种方法可以随机生成）取两组牌作为你的底牌。思考一下我们之前讨论过的内容：
    - 这个两对的范围是什么？
    - 你用这手牌在面对单个对手开池加注时会如何应对？
    - 在有 2 名或更多玩家的单次加注底池中，你会如何应对？
    
    尽量保证你的答案准确，直到你满意为止。
    
3. 加入牌桌，尝试运用我们目前学到的所有两对（以及所有其他）概念。对于两对，请记下
本次练习中对你有用或没用的概念。

## 总结

- 将你的两对划分为顶端、中端、底端三个范围。
- 有利位置 vs. 不利位置的打法。
- 在某些情况下找到正确的玩法。
